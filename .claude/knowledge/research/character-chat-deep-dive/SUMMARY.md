## 2. 교차 패턴 (여러 앱에서 반복 확인된 것)

1. **기억이 이 카테고리의 최상위 과금 지점이자 신뢰 계약.** CHAI·Talkie·PolyBuzz 모두 VOC 1위가 메모리. PolyBuzz는 영구 메모리를 $29.90에, Kindroid는 컨텍스트 크기를 애드온으로 판다. 이기는 쪽은 기억을 **보이게·편집 가능하게**(Kindroid Backstory, Paradot Memory Tab) 만든 쪽이고, 최악의 실패는 **일방적 메모리/인격 교체**(Replika 2.0 리빌드, ERP 제거 "lobotomy")다.
2. **채팅 이력 소실 = 전 앱 공통 최대 분노 포인트.** 대화 기록이 곧 관계 자산. 컴패니언형 리텐션의 본질은 "떠나면 잃는 것"에서 나온다.
3. **이주 선택 기준은 검열 수위 > 메모리 > 무료 한도** 순 (레딧 이주 담론). 2025-11 Character.AI 미성년 금지 이후 시장은 "18+ 선명화 vs 패밀리 프렌들리"로 양극화 중.
4. **관계 몰입이 수익 코어인 동시에 규제 표적.** 한국(국감·게임위), 미국(FTC 조사, 캘리포니아 SB 243), EU(이탈리아 Garante) 모두 동일 지점을 때린다. Tolan처럼 리스크를 디자인으로 제거(비인간+로맨스 금지)하고도 ARR $12M이 가능함이 반례.
5. **UI 강제 개편은 지뢰.** Character.AI 공식 롤백, Kindroid V2 대반발 — 감정 이입된 사용자층은 인터페이스도 관계의 일부로 여긴다.
6. **관계 진전과 결제의 결합은 양날.** 러비더비는 성인 대화를 관계 게이트 뒤에 둬 성공했지만, Paradot은 감정 깊이를 마일스톤 뒤에 게이팅해 "그라인딩" 불만이 1위 — 진전 자체를 팔면 역효과, 진전이 열어주는 **콘텐츠**를 팔아야 한다.
7. **웹 결제 우회는 공공연한 전략.** 제타는 인앱 대비 40% 싼 웹 결제를 노골 유도해 스토어 수수료 회피.

## 3. 수익화 모델 스펙트럼

```
광고 의존(무료 무제한)          하이브리드                    관계-게이트 종량제
PolyBuzz ──── 제타(광고+피스+패스) ──── Talkie(가챠+구독+광고) ──── 크랙(재화+티어링) ──── 러비더비(잼+관계게이트)
                                                                          구독 단일: CHAI·Replika·Kindroid·Nomi·Tolan
```
- 아시아권(제타·크랙·러비더비·Talkie)은 **소모성 재화·가챠**가 코어, 서구 컴패니언은 **구독** 중심.
- 고래 전략의 극단: 러비더비 월 99,000원 구독, 크랙 유료 ARPU $17.

## 4. 벤치마킹 우선순위 (신규 진입자 관점)

1. **러비더비** — 팀 25명으로 아시아 1위: 명확한 ICP(여성향) + 관계 게이트 결제 + 40개 언어 현지화. 자본 없이 이기는 법의 교본.
2. **Tolan** — 규제 리스크를 디자인으로 제거하고 4개월 ARR $12M: 비인간 캐릭터, 친구 초대로 트라이얼 연장, 기억의 시각화("개인 행성").
3. **제타** — 무료 무제한(광고)으로 사용시간을 독점한 뒤 위로 파는 3층 사다리 + UGC 공급비 0 구조.
4. **PolyBuzz** — 자체 모델 없이 TikTok 바이럴+UGC만으로 월 4,280만 방문: 최소 자본 검증 경로.
5. **Kindroid** — 5인 팀 자체 LLM + 화이트박스 기억: 기억을 기능이 아닌 신뢰 계약으로 파는 법.

## 5. 원문(1차 리서치) 대비 갱신·정정된 사실

- 러비더비 "해외 95%"는 매출이 아닌 **이용자 기준**(대만·태국·일본 중심).
- CHAI는 "Character.AI와 유사한 UGC 플랫폼" 이상 — 자체 LLM 메시로 ARR $80M·밸류 $2.4B의 모델 회사.
- Talkie는 2024-12 미국 앱스토어 삭제 → 2025-02 "Talkie Lab"으로 복귀한 전력이 있음(스토어 리스크 실증 사례).
- 제타는 이미 일본 엔터형 AI 앱 1위(월 매출 11억)로, "한국형 c.ai 경쟁사"를 넘어 일본 시장 검증 완료 단계.
- Replika는 성장 스토리가 아니라 **하락세 + 반면교사**(두 차례의 자충수 churn 이벤트)로 읽는 게 정확.
