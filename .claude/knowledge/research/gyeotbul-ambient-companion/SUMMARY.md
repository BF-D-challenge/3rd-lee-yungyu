# 곁불 리서치 통합 요약 (2026-07-12)

> PRD: `docs/prd/b2c-ideas/03-곁불-prd.md` · 상세: 같은 폴더 `COMPETITORS.md`, `COMMUNITY.md`
> 시간 제한(8분) 하에 수집 — 미수집 항목은 각 파일에 명시.

## 핵심 발견

1. **"조용한 동반" 수요는 실재하지만 전부 과업(생산성) 맥락에서 검증됨.** Focusmate($6.99~9.99/월, 150+개국, 수백만 세션)가 유일하게 결제까지 검증된 동반 모델이고, 레딧 r/ADHD·트위터(밈 1.6만 좋아요)의 body doubling 수요도 "일 시작하기"가 목적. **"밤에 친한 사람과 그냥 쉬기"에 돈을 낸 사례는 발견하지 못함.**
2. **친밀한 소수와의 저압적 존재 신호는 초대형 수요** — Locket 8,000만 DL·DAU 900만. 그러나 연매출 ~$1,350만으로 ARPU가 극히 낮고, Widgetable도 무료 21위 vs 매출 99위. **존재감은 성장은 시키지만 결제는 잘 안 시킨다**는 패턴이 일관됨.
3. **결제가 성립한 지점은 존재감 자체가 아니라 부가물**: Focusmate=세션 무제한(생산성), Forest/열품타=재화·편의, Widgetable=대기 스킵·꾸미기, Sky=시즌 콘텐츠·선물. 곁불의 "불씨 7일 유지" 페어 패스는 이 중 어디에도 대응물이 없는 미검증 과금 지점.
4. **Sky의 침묵+공동 의식 설계는 대규모 검증됨**(캔들 의식→관계 해금, 3인 패스 반값 = 이타적 과금). 곁불의 "장작 의식"과 "구매자 1명이 페어 전체 해금" 구조의 직접 근거. 단 Sky는 거대 콘텐츠 월드가 체류를 담보.
5. **직접 경쟁자는 공백**: "친구와 동시 접속하는 가상 모닥불" 앱은 존재하지 않음(혼자 트는 앰비언스 앱뿐). 한편 모닥불+동반 콘텐츠(유튜브 study with me/ambience)는 3,000만 조회급 — 무료 비실시간 대체재가 이미 포화.

## PRD 가정 대조표

| PRD 가정 / Kill 기준 | 판정 | 근거 |
|---|---|---|
| **V: 대화·콘텐츠 없이 친한 사람의 실시간 존재+의식 1개가 의미 있다** | **부분 지지** | body doubling(레딧·트위터 바이럴)과 Locket DAU 900만이 "낮은 압력의 존재 신호" 수요 증명. 단 모두 과업 맥락 또는 비동기 — "친한 2인·실시간·휴식" 조합의 직접 증거는 없음 |
| **U: 초대 링크·두 자리·연결 상태·장작 행동을 설명 없이 이해** | **불명(간접 지지)** | 리서치 범위 밖. 단 r/ADHD VOC의 "상대 무반응 → 오히려 외로움"은 연결 상태 명시가 필수라는 PRD 안전장치를 지지 |
| **Vi: 첫 세션 후 페어 중 1명이 7일 영속성에 3,900원 지불** | **반박 우세** | 최대 위험 적중. 존재감형 제품(Locket·Widgetable)은 결제 전환이 구조적으로 약하고, 결제된 동반(Focusmate)은 생산성 명분이 있었음. "불씨 유지"라는 감정 영속성 과금의 선례 없음. 레딧 VOC에도 유료 저항 명시. Sky의 "선물/페어 반값" 이타 과금이 유일한 우회 근거 |
| **F: Supabase Realtime로 2인 상태 동기화 가능** | **지지(리서치 외 일반론)** | 2인 한정·재연결 TTL 설계는 기술적으로 보수적. 이번 수집 범위에서 반례 없음 |
| **성장: 초대받은 사람이 새 호스트가 되는 비율이 관건** | **지지(구조적 우려 포함)** | Locket·썸원 모두 초대 기반 성장 검증. 단 둘 다 N명 초대 가능 — 곁불은 호스트당 초대 1명이라 K<1 구조라는 PRD 자체 경고가 데이터와 부합 |
| **리텐션 Kill: 7일 내 2회차 세션** | **불명** | 실시간 2인 동시 접속 리텐션 데이터는 어떤 유사 서비스에서도 공개 안 됨. Focusmate는 예약+매칭 풀로 시간 조율 문제를 풀었지만 곁불은 이 해법 사용 불가 → 1주차 실험에서만 확인 가능 |

**종합**: 가치(V) 가설은 실험할 가치가 있으나, 수익(Vi) 가설은 업계 선례가 일관되게 반박 — PRD의 "0결제 시 즉시 Kill" 장치가 타당하며, 결제 지점을 "불씨 유지"에서 Sky형 "상대를 위한 선물·페어 반값" 프레임으로 바꿔 실험하는 것이 승산이 더 높음.

## 벤치마킹 시사점 (5)

1. **결제 명분을 붙여라**: 침묵 동반 단독으로는 과금 선례가 없다. Focusmate처럼 "함께 있으면 ○○이 된다"(휴식의 질, 관계 유지)라는 성과 서사를 결제 문구에 심거나, 1주차 실험에서 "존재감 vs 성과 명분" 두 결제 문구를 A/B로 물어라.
2. **Sky의 이타 과금을 이식하라**: "내 불씨 유지"(자기 소비)보다 "상대에게 7일의 밤을 선물"(이타 소비)이 검증된 프레임. 구매자 1명이 페어를 여는 PRD 설계와 정확히 호환된다.
3. **시간 조율이 최대 병목** — Focusmate는 낯선 사람 매칭 풀로, Locket은 비동기로 풀었다. 곁불은 둘 다 못 쓰므로 1주차에 "약속 잡기 실패율"을 별도 지표로 기록하고, 실패 시 '불씨 남기기' 같은 최소 비동기 신호(위젯형)를 피벗 후보로 준비하라.
4. **무반응 상태는 외로움을 증폭시킨다**(r/ADHD VOC). 재연결·자리 비움 표시를 흐리게 처리하지 말고, 상대가 나가면 즉시 정직하게 보여주는 PRD 방침을 유지하라.
5. **무료 대체재 포화를 인정하라**: 모닥불 앰비언스는 유튜브에서 3,000만 조회 무료 제공. 곁불의 유일한 차별점은 "상대가 누구인지 아는 실시간성"뿐이므로, 데모·랜딩에서 불의 예쁨이 아니라 **두 번째 의자가 채워지는 순간**을 팔아야 한다.
