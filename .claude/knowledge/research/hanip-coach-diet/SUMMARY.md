# 한입코치 리서치 SUMMARY

> 수집일 2026-07-12 · PRD: `docs/prd/b2c-ideas/02-한입코치-prd.md` · 상세: `COMPETITORS.md`, `COMMUNITY.md`
> 경쟁사 웹 딥다이브 + 레딧(rdt)·트위터(twitter CLI)·유튜브(yt-dlp) 수집 완료. 개별 미확인 항목은 각 파일 말미에 명시.

## 핵심 발견 5

1. **사진 칼로리 수요는 증명 끝 — 그리고 흡수·신뢰 붕괴가 동시 진행**: Cal AI 2025년 매출 ~$30M → $50M+ ARR 연환산, 1,500만 DL, **2026-03 MyFitnessPal에 매각**($100M설) 직후 **2026-04 기만적 결제 화면으로 앱스토어 일시 퇴출**. 국내에도 클론 다수(SnapEat·Calsee 등). "사진→숫자"만으로는 차별화 불가, "행동 개입 + 투명 과금"이 빈 자리.
2. **정확도 회의론이 커뮤니티·리뷰 생태계의 주류 정서**: r/loseit 저울 실측(⬆2,474) — 최고 모델도 ~20% 오차, 병목은 부피 추정, 혼합 요리·비서구 음식(한식) 30~50% 오차. "False precision is more dangerous than admitted uncertainty" → PRD의 "AI 후보+사용자 확인"은 업계 실패 지점을 정확히 우회.
3. **"빡센 코치" 수요의 직접 증거 확보**: r/loseit "Please bully me into losing this weight"(⬆347·댓글 437), 유료 책임감 서비스 지불 의사 발화, tough love 글 자체가 소비됨(⬆690). 국내 1위 필라이즈는 "다정한 AI 고냥이 🥺" 톤 → **단호함 포지션은 국내 공백.** 직접 선례도 존재: Coach Sal("brutally honest" + 톤 강도 사용자 선택).
4. **안전 리스크는 실존 선례 + 양방향**: CARROT Fit은 NEDA가 바디셰이밍 사례로 지목, Noom은 ED 스크리닝 부실·$62M 자동갱신 합의. 연구로는 ED 환자의 73%가 "MFP가 내 ED에 기여"(Levinson 2017). 동시에 **안전필터 과잉 발동은 정상 사용자를 모욕해 이탈시킴**(Claude 사례 ⬆104) — 스크리닝 강화와 임계값 정교화 둘 다 필요.
5. **독설의 대중화 형태는 "권위자가 웃으면서 때리는 팩폭"**: 유튜브에서 순수 독설 단독은 틈새(수백~수천), 쓴소리+권위/유머는 수십만~백만(유퀴즈 366만, 키다리형 36.1만). CARROT도 화제성 대비 리텐션 제품화 실패. 혼내는 앱들의 수렴 진화 = **톤 강도 선택 옵션** — PRD의 "차분하게/단호하게" 설계와 일치.

## PRD 가정 대조표 (VUVF + Kill 기준)

| 가정 | 판정 | 근거 |
|---|---|---|
| **V**: 포기자는 중립 숫자보다 안전한 단호함에서 더 행동한다 | **부분 지지** | tough love 수요 발화 실재(⬆347·댓글 437 등) + 국내 톤 공백 + Coach Sal 선례. 단 "수요 발화"≠"행동 변화 효과" — CARROT 리텐션 실패·독설 단독 콘텐츠 저조가 반대 신호. 1주차 교차 실험으로만 최종 판정 |
| V 하위: 기록 포기 문제의 실재 | **강한 지지** | MFP RCT 기록률 1주차 68%→12주차 21%. 포기 3유형(피로/이벤트 붕괴/강박 자기방어) 대량 업보트 VOC. "작심삼일" 국내 트윗(❤823) |
| **U**: 사진→확인→행동 선택 60초 | **부분 지지** | 사진 1장 UX 대중 검증. 단 실측상 양(부피) 확인이 필수인데 이 마찰 추가 후 60초 완결은 미검증. 한식 오차가 더 커 국내 부담 가중 |
| **Vi**: 분석 1회 아닌 7일 책임 루프에 4,900원 | **불명 (약한 지지)** | 사람 코치 WTP 월 3~10만원(다노)·유료 accountability 지불 의사 발화·클론도 $1-5k MRR. "AI 책임 루프" 지불 선례는 없음(Coach Sal 지표 미확인) — 1주차 실결제가 유일한 검증. Cal AI·Noom 결제 신뢰 붕괴는 투명 과금(비갱신 7일 패스)의 차별화 기회 |
| **F**: 사용자 확인 후 안전한 행동 제안 자동화 가능 | **부분 지지** | 음식 식별은 잘 되고 양 추정만 병목 → "확신도 낮은 항목만 묻기"와 정합. 위험 문장 0건 보장은 미검증 |
| 최대 위험 1: 독설→수치심 아닌 행동 | **판정 불가 — 양쪽 증거 공존, 위험 실증** | 수요(빡센 코치 요청)와 위험(ED 환자 73% "앱이 기여", NEDA 지목 선례, 죄책감·이분법 UI의 ED 촉발 연구) 동시 실재. **톤 Kill 기준 유지 + 자기보고 전에 "다음 사진 재제출률" 하락으로 회피 행동 조기 감지 권장** |
| 최대 위험 2: 무료 대안 많은데 4,900원 지불 | **불명** | 실결제 실험 외 판단 불가. 무료 ChatGPT 코치 활용 콘텐츠(33만+ 조회 "It Beat EVERY Paid App")가 실질 경쟁자로 부상 중 |
| 섭식장애 안전 설계(제외 대상·중단·금지 문장) | **지지 (강화 권장)** | 방향은 CARROT·Noom 실패를 정확히 회피. **과거 ED 이력 스크리닝**(Noom도 빠뜨림) + 과잉 발동 방지 임계값까지 해야 완결 |
| 선행 지표 "수치심·불안 보고 10% 미만" | **측정법 보완 필요** | 수치심은 자기보고보다 회피(사진 미제출)로 먼저 나타남 — 행동 지표 병행 필수 |

## 벤치마킹 시사점 (5)

1. **포지셔닝은 "단호함"보다 "복귀"를 앞세우고, 톤은 "권위자가 웃으면서 때리는 팩폭"으로** — 순수 독설은 획득·유지 모두 약하고 fat shaming 프레임 리스크만 키움. 톤 강도 선택(PRD 설계)은 업계 수렴 진화와 일치하므로 유지.
2. **정확도 회의론을 차별화 근거로 역이용**: "숫자 말고 행동을 줍니다" — 칼로리 숫자는 실측 공격을 받지만 "다음 끼니 행동 1개"는 검증 공격이 불가능한 가치 제안.
3. **온보딩에 과거 ED 이력 문항 + 안전필터 임계값 정교화** — Noom의 실패 지점과 과잉 발동의 역효과(정상 사용자 모욕)를 동시에 회피.
4. **투명 과금 자체를 셀링 포인트로** — Cal AI 퇴출·Noom $62M 합의로 카테고리 결제 신뢰 공백. 비갱신 단건 7일 패스는 구조적으로 유리, 결제 화면에 갱신 없음 명시.
5. **배포는 Cal AI 플레이북(오가닉 선행→마이크로 인플루언서 리테이너)** + 영상 문법은 "AI 코치한테 한 달 팩폭 당해봤다 → n kg" 1인칭 실험담. 국내 "피드백/코치" 프레임 영상은 공백이라 선점 가능.
