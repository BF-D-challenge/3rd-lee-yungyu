# 배치 003 요약

## 결과

- 검토: 20개
- UX 합의 통과: 15개
- 한국 하드 게이트 포함 통과: 12개
- 탈락: 7개
- 보류: 1개

## 가장 강한 시장 신호

- 프리랜서 미수금: 국내 프리랜서 매니저 제품이 견적·입금 확인·미수금 알림을 제공해 반복 문제를 확인
- 중고거래: 고가 전자제품 직거래 전 상태·일련번호·흠집 확인 필요
- 전월세 날짜: 법제처 생활법령에서 계약 종료 통지 기간과 갱신 관련 원문 확인 가능
- 발표·말하기: 국내외 스피치 코칭·발표 타이머 제품이 존재하며 반복 연습 수요 확인
- 총무·공동구매: 회원·주문 명단과 입금 내역 대조가 반복되는 운영 문제

## 강한 탈락 신호

- 관리비·전기요금: K-apt, 아파트아이, 한전ON과 한전 AI 알림이 이미 해결
- 자동차 정비: 마이클·카닥이 기록·교체주기·견적 비교를 해결
- 냉장고·세탁: 영수증 OCR 냉장고 앱과 라벨 스캔 앱, iPhone 기본 인식이 존재
- 짐검사: 인천공항·항공사 공식 정보와 무료 반입물품 앱 존재
- 진료메모: 국내 동일한 환자 중심 녹음·요약·가족 공유 제품 확인

## 위험

- 계약·근로·의료 영역은 판정이나 처방이 아니라 원문 표시·사용자 확인·기록 정리에 한정
- 금융·계약·의료 파일은 기기 내 처리, 자동 마스킹, 원본 미저장 가설을 실제 구현으로 검증
- CSV 의존 후보는 사용자가 실제로 파일을 내려받을 수 있는지 컨시어지 테스트 필요
