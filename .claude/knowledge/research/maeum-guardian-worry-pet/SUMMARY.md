# 마음수호신 리서치 요약 (2026-07-12)

> PRD: `docs/prd/b2c-ideas/05-마음수호신-prd.md` · 상세: `COMPETITORS.md`, `COMMUNITY.md`
> ⚠ 시간 제한으로 Rootd·How We Feel·Worry Watch·점신·헬로우봇·마보/코끼리 상세는 미수집(시간 제한). Finch·포스텔러·커뮤니티 VOC는 실측 완료.

## 핵심 발견

1. **Finch가 "펫 애착→재사용·결제"를 대규모로 증명** — VC 없이 ARR ~$30M, 월 매출 추정 ~$2.9M, Play 1,000만+ DL. 단 **결제 동기는 위로 '기능'이 아니라 코스메틱+감사/후원**이며 핵심 케어 도구는 전부 무료. ([Sparrow](https://blog.sparrowapps.io/p/finch-how-a-self-care-app-hit-30m-arr-without-vc-money), [Sensor Tower](https://app.sensortower.com/overview/1528595748?country=US), [Plus 가격](https://help.finchcare.com/hc/en-us/articles/38755205001869-Finch-Plus-Pricing))
2. **포스텔러 = 3,900원 일회성 리포트 가설의 국내 선례** — 누적 900만·월 140만 유저, 연 매출 100억+, 콘텐츠 5%만 유료·**구독 아닌 코인 단건 결제**로 흑자. ([한국일보](https://www.hankookilbo.com/news/article/A2023051514260005915), [ZDNet](https://zdnet.co.kr/view/?no=20250203104315))
3. **무료 AI 우회가 최대 위협** — "만세력 캡처→ChatGPT 풀이" 트윗 2,835 좋아요 바이럴. 텍스트 풀이·위로문은 이미 무료 대체됨 → 유료 리포트는 **본인의 7일 행동·감정 데이터 패턴**(GPT가 복제 불가)이어야 생존.
4. **r/finch VOC: 애착의 이면** — 데이터 유실=애착 붕괴(devastated), 무료 보상 축소·AI 광고에 대규모 반발(1,082↑), 커뮤니티가 스스로 "천천히 해도 돼"(1,491↑)로 의무감을 경계 → PRD의 비의존·무죄책감 설계는 유저 정서와 정확히 공명.
5. **디지털 걱정인형 직접 경쟁자 부재** — 레딧·트위터에서 "걱정을 맡긴다" 제품 담론 자체가 희박. 선점 기회인 동시에 검증된 직접 수요 증거도 없음(1주차 실험이 그래서 필수).

## PRD 가정 대조표 (VUVF + Kill 기준)

| 가정 | 판정 | 근거 |
|---|---|---|
| **Value**: 펫+의식이 일반 AI 위로보다 낫다 | **부분 지지** | Finch 애착·리텐션 실증(r/finch 감사 포스트, 3년 결제). 단 Finch 애착은 장기 육성·스트릭·커뮤니티 산물 — **90초 단발 의식만으로 애착이 생기는지는 불명** → 1주차 A/B 필수 |
| **최대 위험 1**: 펫 호감이 재사용·결제로 안 이어질 수 있다 | **Finch가 절반만 반증** | 재사용·결제 전환은 증명됨. 그러나 결제는 '위로 기능'이 아니라 꾸미기·감사에서 발생 → **마음수호신의 '리포트 결제' 가설을 Finch가 직접 지지하진 않음** |
| **Viability**: 7일 패턴 리포트에 3,900원 낸다 | **문법은 지지 / 내용물은 위험** | 포스텔러 코인 단건 결제로 연 100억(사주 유저 소액·단건 지불 문법 검증). 반면 무료 AI 우회 바이럴 → 리포트가 '풀이 텍스트'면 대체당함, '내 데이터'면 방어 가능 |
| **Usability**: 불안 순간에 한 줄+90초 완수 | **불명** | 외부 선례 없음. Kinder World '2분 의식'(100만+ DL, PRD 기재)이 간접 지지 수준 |
| **Feasibility**: 안전분류+생성문+3상태 4주 구현 | **불명(리서치 범위 외)** | — |
| **Kill: 죄책감·의존 유발 시 중단** | **설계 방향 지지** | Finch 유저 스스로 스트릭 압박 경계 문화(1,491↑), 보상 축소에 반발 → 비의존 설계가 차별점이자 안전판 |
| **K(공유) 가설** | **불명** | Finch 성장은 TikTok 오가닉+커뮤니티 — 수호동물 결과 공유 연쇄는 국내 미검증 |

## 벤치마킹 시사점 (5)

1. **핵심 위로는 영원히 무료, 결제는 '애착의 표현'으로 설계** — Finch 공식: 케어 도구 무료 + 코스메틱/후원 결제. 리포트 단품과 병행해 '수호동물 꾸미기/후원' 결제 축을 2안으로 준비할 것.
2. **결제 UX는 구독이 아니라 단건** — 포스텔러 코인 모델이 한국 운세 유저 문법. 3,900원 일회성 유지가 맞다.
3. **리포트의 방어선은 데이터** — GPT로 복제되는 일반 풀이 금지. "당신이 실제 기록한 7일의 감정·행동 패턴"만이 유료 가치.
4. **정원 데이터 보존 = 애착 제품의 생명선** — Finch 데이터 유실 절망 사례. 백업·복구를 3주차 Supabase 설계에 1급 요구사항으로.
5. **콘텐츠 문법은 '함께 있는 시간'** — Finch 마케팅은 기능 설명이 아니라 birb와의 lofi 6시간·브랜드 필름. 수호동물 공유물도 '소개 카드'보다 '함께 쉬는 장면'이 퍼질 확률 높음.
