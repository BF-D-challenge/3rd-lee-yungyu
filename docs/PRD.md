# 오늘 해볼까 — PRD 한 장 요약

> 기준 명세: [최종 PRD v5](prd/오늘-해볼까-prd.md) · 2026-07-12

## 제품

검증된 제품의 작동 방식과 한국 사용자 맥락을 미리 카드로 생성·검수해 두고, 네 장을 즉시 조합해 80~99%는 남기고 한 가지만 바꾼 아이디어를 전달한다. 사용자는 결과를 공유해 전체 제작 문구를 열고, 지인에게 익명 응원을 받는다.

## 네 장

1. 검증된 원본
2. 돈 낼 사람
3. 필요한 순간
4. 한 끗 변화

취향 조사와 고정은 사용하지 않는다. 전체 다시 뽑기, 한 장 바꾸기, 직접 입력을 제공한다.

현재 앱에는 원본 시나리오 52개와 원본당 27개, 총 1,404개 런타임 조합이 사전 탑재돼 있다. 통합 원본 장부와 52/52 연결됐으며 장기 목표는 원본 100개·2,700조합이다.

## 소셜 루프

네 장 뽑기 → 결과 확정 → 제작 문구 앞부분 확인 → 공유 → 전체 제작 문구 열림 → 익명 응원 → 모닥불의 빈자리 채움과 불꽃 성장 → 받은 사람도 네 장 뽑기

응원 전은 유령과 점선 나무 의자, 응원 후는 사람과 실제 나무 의자다. 성장은 장작 층이 아니라 불꽃 크기로 표현한다.

## 성장과 수익

- 공유 보상: 전체 제작 문구를 무료로 연다.
- 이름 확인: 실제 응원 도착 후, 동의한 발신자 이름을 30일보다 일찍 확인하려는 990원 가짜 문이다.
- 가장 중요한 지표: 아이디어 확정 후 7일 안에 실제 익명 응원을 한 개 이상 받은 비율.

## 이번 범위

검증된 원본 기반 네 장 뽑기, 쉬운 결과 설명과 제작 문구, 공유 해제, 매일 한 장 익명 칭찬까지 구현한다. 앱 구현 체크리스트는 [docs/TASK](TASK.md), 사전 생성 카드 확장은 [루트 TASK](../TASK.md)를 따른다.
