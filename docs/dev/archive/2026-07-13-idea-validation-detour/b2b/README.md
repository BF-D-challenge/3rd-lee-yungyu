# B2B 무코드 검증 키트

업데이트: 2026-07-13  
대상: `B09-01 첫결과 구조대`, `LC11 펫시터 인계팩`

## 운영 원칙

- 이 폴더는 **실제 외부 발송·결제 전 준비물**이다. `[링크]`, `[날짜]`, `[운영자]`는 실행 직전에만 채운다.
- `좋아요`, `쓸 것 같아요`, 무료 신청은 수요 증거로 세지 않는다. 표시된 가격의 실제 결제 또는 전액 환불 가능한 유료 예약만 결제 신호다.
- 아직 자동화되지 않은 Wizard-of-Oz 실험임을 랜딩과 결제 전에 명시한다.
- 결과를 보장하지 않는다. 관찰된 사실과 제공 범위만 말하고, 환불 조건을 결제 전에 보여준다.
- 이름 대신 `P001` 같은 실험 ID를 쓴다. 전화번호·주소·계정 비밀번호·출입 비밀번호·민감한 원본은 기록하지 않는다.
- 인터뷰 메모와 운영 데이터는 분리한다. 실험 종료 또는 철회 후 14일 안에 삭제한다.
- 각 후보의 `decision-scorecard.md`를 사전에 고정하고, 결과를 본 뒤 기준을 낮추지 않는다.

## 비교할 공통 행동

1. 최근 30일 또는 가장 최근 영업 1건의 실제 문제가 있었는가
2. 현재 해결에 시간·돈을 쓰고 있는가
3. 수기 결과물을 실제 업무에 사용했는가
4. 제시 가격을 본 뒤 결제했는가
5. 결제하지 않았다면 첫 거절 이유가 무엇인가
