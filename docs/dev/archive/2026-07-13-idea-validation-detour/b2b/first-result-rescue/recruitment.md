# 첫결과 구조대 — 강사 5명 모집 메시지

공통 타겟: 매달 10~100명의 유료 강의·코칭·챌린지를 혼자 운영하고, 결과물을 카카오톡·폼·노션 등 강의 밖에서 받는 운영자.

## 1. 커뮤니티 게시용

제목: 유료 수강생의 `첫 결과물 미제출`을 직접 챙기는 강사님 5분을 찾습니다

안녕하세요. 유료 코호트에서 진도율이 아니라 **결제 후 7일 안에 첫 결과물을 내지 못한 수강생**을 어떻게 찾고 계신지 조사하고 있습니다.

최근 운영한 코호트의 실제 흐름을 20분 동안 여쭙고, 원하시면 결제자 임시 ID 명단을 기준으로 D+7 미제출자 목록을 수기로 만들어드립니다. 인터뷰는 판매 전화를 위한 것이 아니며, 이름·연락처·수강생 결과물 원본은 받지 않습니다.

- 대상: 월 10~100명 유료 프로그램 운영자
- 시간: 20분
- 모집: 5명
- 신청: `[신청 링크]`

현재 수기 파일럿이며 자동화 제품으로 과장하지 않습니다.

## 2. 개인 DM용

안녕하세요, `[상대가 운영한 프로그램명]`을 보고 조심스럽게 연락드렸습니다. 강의 진도와 별개로 과제·사진·파일 같은 첫 결과물을 외부 채널에서 받으시는 것 같았습니다.

최근 코호트에서 **결제했지만 D+7까지 첫 결과물을 못 낸 사람을 어떻게 찾았는지** 20분만 실제 사례 중심으로 여쭤봐도 될까요? 운영 방식이 맞으면 임시 ID 명단으로 미제출자 목록을 수기 제작해 드립니다. 수강생 이름·연락처·원본 결과물은 받지 않습니다.

가능하시면 `[일정 링크]`에서 편한 시간을 골라주세요. 불편하시면 답하지 않으셔도 됩니다.

## 3. 지인 소개 요청용

혹시 매달 10~100명 규모의 유료 강의·코칭을 혼자 운영하면서, 과제나 첫 결과물을 카톡·폼으로 따로 받는 분 한 분만 소개해주실 수 있을까요?

제가 조사하는 건 “진도율”이 아니라 **돈을 내고도 첫 결과물 없이 7일을 넘긴 수강생을 찾는 일**입니다. 20분 인터뷰 5건만 진행하며, 판매 약속이나 개인정보 제공을 요구하지 않습니다.

소개하실 때 아래 문장만 전달해 주세요.

> `[운영자]`가 최근 실제 코호트 운영 방식을 20분 듣고 있습니다. 자동화 제품은 아직 없고, 원하면 임시 ID 기준 D+7 미제출자 목록을 수기로 만들어준다고 합니다. 관심 있으면 `[연락 링크]`로 직접 연락해주세요.
