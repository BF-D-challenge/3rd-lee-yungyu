# 펫시터 인계팩 — 독립 펫시터 5명 모집 메시지

공통 타겟: 중개 앱 밖에서 카카오톡·문자로 보호자와 직접 예약하고, 주 1회 이상 돌봄 결과를 직접 보고하는 독립 펫시터.

## 1. 커뮤니티 게시용

제목: 카톡으로 보호자 요청과 돌봄 보고를 직접 관리하는 펫시터 5분을 찾습니다

독립 펫시터가 최근 돌봄에서 보호자 요청을 다시 확인하고, 식사·산책·배변·특이사항을 보고하는 실제 흐름을 조사하고 있습니다.

20분 인터뷰 후 원하시면 보호자 앱 설치 없이 **요청별 완료 여부와 사진을 한 링크로 보여주는 수기 인계팩**을 한 건 만들어드립니다.

- 대상: 주 1회 이상 직접 예약·보고하는 독립 펫시터
- 시간: 20분
- 모집: 5명
- 신청: `[신청 링크]`

주소·출입 비밀번호·보호자 연락처·결제 정보는 받지 않으며, 건강 판단이나 투약 조언을 제공하지 않습니다. 현재 자동화되지 않은 수기 파일럿입니다.

## 2. 개인 DM용

안녕하세요, `[상대의 공개 활동/서비스]`를 보고 조심스럽게 연락드렸습니다. 중개 앱보다 카톡으로 보호자 요청을 받고 돌봄 결과를 직접 보고하시는 흐름을 조사하고 있습니다.

가장 최근 돌봄 한 건에서 요청을 확인하고 사진·특이사항을 보낸 과정을 20분만 여쭤봐도 될까요? 원하시면 실제 개인정보 대신 예시 정보로 **보호자 요청 → 완료 결과 링크**를 수기로 만들어드립니다.

주소·출입 비밀번호·연락처는 받지 않습니다. 가능하시면 `[일정 링크]`에서 시간을 골라주세요. 불편하시면 답하지 않으셔도 됩니다.

## 3. 지인 소개 요청용

혹시 카카오톡으로 보호자와 직접 예약하고, 주 1회 이상 식사·산책·배변·사진 보고를 보내는 독립 펫시터 한 분을 아시나요?

최근 실제 돌봄 보고 흐름을 20분 듣고, 원하면 보호자 가입 없는 수기 확인 링크를 한 건 만들어보는 조사입니다. 판매 약속이나 주소·출입 정보 제공을 요구하지 않습니다.

아래 문장을 그대로 전달해 주세요.

> `[운영자]`가 독립 펫시터의 가장 최근 요청·보고 흐름을 20분 듣고 있습니다. 자동화 제품은 아직 없고, 개인정보 없이 수기 인계 링크를 한 건 만들어본다고 합니다. 관심 있으면 `[연락 링크]`로 직접 연락해주세요.
