## 입력 1개 → 결과 1개

- 입력: 실제 이력서의 성과 문장 1개
- 결과: `문장 · 내 역할 · 수치 기준 · 안전한 근거 링크`가 한 장에 담긴 면접용 증거 카드 1장

예시:

```text
입력: 신규 온보딩 개선으로 4주차 전환율 18% 상승

결과 카드
- 내 역할: 이탈 구간 분석, 실험안 작성
- 수치 기준: 개선 전후 동일 코호트의 4주차 전환율
- 근거: 공개 출시 공지 / 숫자를 가린 차트 / 본인 작성 회고
- 30초 답변: 상황 → 내가 한 일 → 측정 결과 → 한계
```

## 왜 ChatGPT·포트폴리오 앱과 다른가요?

- ChatGPT·Claude: 문장을 잘 써주지만 그 문장이 어느 원본과 연결되는지 계속 보관하지 않는다.
- 포트폴리오 앱: 완성된 작업물을 보여주지만 이력서의 주장 한 줄과 수치의 출처를 면접 직전에 찾는 문제는 남는다.
- 경력성과증거팩: **문장과 근거의 연결 상태만 소유한다.** 채용 추천, 합격 예측, 경력 사실 인증은 하지 않는다.

## 회사 기밀은 받지 않습니다

회사 내부 문서, 고객 개인정보, 미공개 코드, 계약서, 원본 매출표는 업로드하지 않는다. 사용할 수 있는 근거는 다음 세 단계뿐이다.

1. 공개 링크: 보도자료, 공개 릴리스, 공개 저장소처럼 누구나 열 수 있는 자료
2. 비식별 근거: 회사명·고객명·계정·절대금액을 가린 본인 소유 캡처
3. 대체 증거: 파일 없이 `측정 시점·비교 기준·본인 역할·회고`만 적고 **본인 진술**로 표시한 메모

근거의 진위를 보증하거나 회사 자료 반출을 권하지 않는다. 애매하면 파일 대신 대체 증거를 사용한다.
