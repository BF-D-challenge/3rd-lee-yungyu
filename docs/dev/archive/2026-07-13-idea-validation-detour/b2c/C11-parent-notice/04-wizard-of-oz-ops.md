# 부모님공문 — Wizard-of-Oz 4주 수기 운영

## 운영 범위

- 참가 가구 10곳, 4주
- 월·목 주 2회 공지 묶음 접수
- 가구당 주 10건 이하
- 운영자가 학교 계정에 로그인하거나 서명·제출을 대신하지 않는다.
- 의료·법률·긴급 안전 공지는 요약하지 않고 원문 확인을 안내한다.

## 4주 추적 계획

| 시점 | 운영 | 측정 |
|---|---|---|
| Week 0 | 자격 확인, 개인정보 교육, 가족 담당자 1명 초대 | 기존 채널 수, 지난 4주 누락, 현재 정리 시간, 결제 |
| Week 1 | 월·목 수기 카드 전달 | 제출 공지 수, 첫 카드 열람, 담당 지정, 완료 체크 |
| Week 2 | 같은 방식 반복, 누락·오분류 수정 | 2주 연속 사용, 가족 2인 사용, 재확인 메시지 감소 |
| Week 3 | 운영자 재촉 없이 자발 제출 여부 확인 | 자발 재사용, 주간 사용률, 미처리 항목 |
| Week 4 | 마지막 카드, 정가 재결제 제안 | 4주 연속 사용, 결제·거절, 계속 쓸 이유 |

4주 연속 사용은 매주 `공지 1건 이상 제출 + 가족 카드 1회 이상 열람 + 완료 체크 1건 이상`을 모두 만족한 가구만 센다.

## 공지 스크린샷 개인정보 규칙

### 참가자가 보내기 전에 반드시 삭제·가림

- 아이 실명, 얼굴, 생년월일, 학생·원생 번호, 반·번호
- 학교·학원 세부 식별정보가 필요 이상으로 드러나는 부분
- 보호자·교직원 이름, 전화번호, 이메일, 주소
- QR·바코드, 계정 ID, 결제번호, 개인별 URL
- 다른 아이와 가정의 정보

결과에는 `첫째/둘째` 같은 별칭, 할 일, 일반화한 마감, 가족 담당, 완료 상태만 남긴다.

### 운영자 삭제 SLA

1. 식별정보 발견: 처리 중단, 원본 즉시 삭제, 참가자에게 재가림 요청
2. 정상 원본: 결과 카드 전달 후 **24시간 안에 영구 삭제**
3. 파생된 할 일 데이터: 파일럿 종료 후 7일 안에 삭제
4. 참가자 삭제 요청: 24시간 안에 원본·파생 데이터 모두 삭제
5. 삭제 시각만 CSV에 기록하며 원문 내용·아이 이름은 적지 않는다.

## 단계별 절차와 메시지

### 0. 가족 코드 발급·동의

가구는 `PN-001` 형식으로만 기록한다. 연락처와 참가 코드를 운영 CSV에 함께 적지 않는다.

```text
[PN-___] 4주 부모님공문 실험에 참여하기 전 확인해주세요.
공지를 보내기 전에 아이·학교·가족을 식별할 수 있는 이름, 얼굴, 번호, 연락처, 주소, QR을 가려주세요. 운영자는 원문을 제출하거나 판단하지 않고 이번 주 할 일만 옮깁니다. 원본은 카드 전달 뒤 24시간 안에 삭제됩니다.
[동의 체크 placeholder]
```

### 1. 월·목 입력 요청

```text
이번 회차 공지를 보내주세요. 가린 스크린샷 최대 5장과 아이 별칭(첫째/둘째)만 받습니다.
긴급·의료·법률·결제 공지는 보내지 말고 기관 원문을 직접 확인해주세요.
마감: 오늘 [시간 placeholder]
```

### 2. 개인정보 사전 검사

이미지를 확대해 금지 항목을 먼저 확인한다. 하나라도 발견하면 내용을 옮기지 않고 즉시 삭제한다.

```text
식별 가능한 정보가 남아 있어 이 이미지는 처리하지 않고 삭제했습니다. 이름·얼굴·반번호·연락처·주소·QR을 가린 뒤 다시 보내주세요. 공지 내용은 별도 기록하지 않았습니다.
```

### 3. 가족 한 장 제작

```text
아이 별칭 | 할 일 | 마감 | 담당 | 완료
첫째 | 준비물 챙기기 | 화 08:30 | 미정 | □
둘째 | 동의 여부 원문에서 확인 | 수 18:00 | 미정 | □
```

- 원문의 의미를 추론하거나 바꾸지 않는다.
- 제출이 필요한 경우 `제출 완료`라고 쓰지 않고 `기관 원문에서 제출 확인`으로 남긴다.
- 모호한 항목은 `[원문 확인 필요]` 표시와 함께 원문 앱 이름만 적는다.

### 4. 전달·원본 삭제

```text
[PN-___] 이번 주 가족 한 장입니다: [가족 링크 placeholder]
담당을 정하고 끝난 항목만 체크해주세요. [원문 확인 필요]는 학교·학원 앱에서 직접 확인해주세요.
받은 원본 스크린샷은 [삭제 예정 시각]까지 삭제합니다.
```

삭제 후:

```text
이번 회차 원본 스크린샷 삭제를 완료했습니다: [삭제 시각]. 가족 한 장에는 아이 별칭과 할 일만 남아 있습니다.
```

### 5. 주간 사용 확인

```text
이번 주 가족 한 장에서 실제로 끝낸 항목 수와 다시 카톡으로 물어본 횟수만 알려주세요.
완료 __건 / 재확인 메시지 __회 / 놓친 항목 __건
```

### 6. Week 4 결제·재사용 확인

```text
4주 동안 가장 줄어든 행동 하나는 무엇인가요? 다음 4주도 계속 쓰려면 가족 패스는 4,900원입니다. [결제 링크 placeholder]
결제하지 않는 이유 하나: 문제 없음 / 학교 앱 충분 / 직접 정리가 빠름 / 가격 / 개인정보 / 가족이 안 씀 / 기타.
```
