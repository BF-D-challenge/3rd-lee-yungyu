# 실험 개인정보·동의 규칙

## 공통 원칙

- 실험에 필요하지 않은 원본은 받지 않는다.
- 실제 이름 대신 인터뷰 ID를 사용한다.
- 주민번호·계좌번호·카드번호·주소·연락처·회사 기밀은 업로드받지 않는다.
- 화면 예시가 목적이면 참가자가 직접 가린 사본을 받는다.
- 수기 결과 제공이 끝나면 원본을 삭제하고 결과 상태만 남긴다.
- 홍보 사례로 쓰려면 실험 참여 동의와 별도의 공개 동의를 다시 받는다.

## 수집 전 고지 문구

> 이 실험은 제품 수요를 확인하기 위한 수기 테스트입니다. 제공한 자료는 결과를 만드는 용도로만 사용하고, 실험 종료 또는 삭제 요청 시 즉시 삭제합니다. 주민번호·계좌번호·카드번호·주소·회사 기밀은 보내지 마세요. 결제 여부와 관계없이 언제든 참여를 중단할 수 있습니다.

## 후보별 금지 자료

| 후보 | 받지 않는 자료 | 허용되는 대체 자료 |
|---|---|---|
| 첫결과 구조대 | 수강생 전화번호·민감 과제 원문 | 가명 명단·제출 여부·비식별 결과 링크 |
| 펫시터 인계팩 | 집 주소·도어락 번호·보호자 연락처 | 가명 반려동물·일반 요청·위치 제거 사진 |
| 경력성과증거팩 | NDA 자료·고객 명단·내부 매출 원장 | 공개 링크·가린 화면·범위화된 수치 |
| 부모님공문 | 아이 실명·학교·반·연락처 | 이름과 학교를 가린 공지 사본 |

## 삭제 기록

원본 삭제 시 `인터뷰 ID / 후보 ID / 삭제 시각 / 삭제 담당 / 남긴 비식별 필드`만 기록한다.
