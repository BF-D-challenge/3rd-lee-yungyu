# docs/bfd-setup — BF.D 세팅 도구 원본 (보관용)

이 폴더는 BF.D 세팅 레포에서 온 **설치 원본**이다. 이 레포에는 이미 설치가 끝났으므로 보관용으로만 둔다.

- `SETUP.md` — 세팅 절차 (⚠️ 문서 안의 `cp skills/…`·`cp templates/…` 경로는 레포 루트가 아니라 **이 폴더 기준**으로 실행)
- `skills/` — 스킬 원본 → 이미 `.claude/skills/`(prd-maker · safe-push · minjung-coach)에 설치됨
- `templates/` — `.gitignore`·`.env.example`·`deploy.yml`(→`.github/workflows/`)·`claude-settings.json`(→`.claude/settings.json`) 원본 + [4주 챌린지 플레이북](templates/4주-챌린지-플레이북.md)

플레이북만 멤버가 직접 읽는 문서이고, 나머지는 운영진 유지보수용이다.
