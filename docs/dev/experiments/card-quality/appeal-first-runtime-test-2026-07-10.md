# 매력 우선 카드 런타임 실험

## 결론

기존 73장을 더 고르게 노출하는 것으로는 해결되지 않는다. 사람이 읽고 바로 장면을 떠올릴 수 있는 10장만 먼저 노출하는 `appeal-first-v1` 코호트를 적용한다.

적용 범위는 뽑기 화면에 보이는 상위 취향 3개가 `로컬`, `가벼운 재미`, `친구 투표`인 사용자다. 뒤에 저장된 보조 취향은 이번 실험의 노출 조건을 깨지 않는다.

## 문제 진단

- 73장 수동 감사 평균은 `21.40/30`, 그대로 유지할 카드는 `0장`이었다.
- 가장 낮은 축은 차별성 `2.58/5`였다.
- 동일 취향 50,000회 기준 일반적인 도구명 카드 노출은 `32.3%`였다.
- 상위 4장이 노출의 `26.2%`를 차지했지만 수동 매력 점수는 `17/30`, `18/30`, `23/30`, `23/30`이었다.
- 카테고리·형태·키워드 점수를 제곱 가중한 기존 선택기가 평범한 카드까지 반복 노출했다.

## 문구 실험

| 라운드 | 평균 | 최저 | 즉시 탈락 |
|---|---:|---:|---:|
| 첫 10장 | 4.30/5 | 3.5 | 1장 |
| 1차 수정 | 4.55/5 | 4.0 | 0장 |
| 최종 수정 | 4.70/5 | 4.5 | 0장 |

최종 10장은 `내 카페 자리`, `첫마디`, `전단 노트`, `오늘 데이트`, `한 권만`, `가게 첫인사`, `운세 한 장`, `우산 여행`, `첫잔 게임`, `커피 한 표`다.

탈락 원인이었던 추상적인 결과 문구와 작위적인 동사를 실제 행동으로 교체했다. 예를 들어 `사진 세 장이 남는다`는 `각자 걸을 거리와 기다릴 시간을 누르면 세 카페에 익명 표가 모인다`로 바꿨다.

## 선택기 실험

100,000회, 취향 `로컬 + 가벼운 재미 + 친구 투표` 고정 결과다.

| 지표 | H5 전체 67장 | H6 검수 10장 |
|---|---:|---:|
| 취향 적합 | 100% | 100% |
| 페어 대비 통과 | 89.6% | 97.9% |
| 같은 형태 | 10.0% | 2.1% |
| 같은 카테고리 | 0% | 0% |
| 최대 단일 카드 점유 | 12.3% | 13.5% |

H6는 10장을 균등 추첨하고 두 번째 카드는 최대 세 후보 중 첫 카드와 가장 덜 겹치는 카드를 고른다.

## 브라우저 확인

실제 연속 세 라운드에서 다음 페어가 노출됐다.

1. `커피 한 표` / `가게 첫인사`
2. `첫마디` / `첫잔 게임`
3. `우산 여행` / `오늘 데이트`

완료 이벤트에서 `selection_variant=appeal-first-v1`, 두 카드의 취향 적합, 페어 대비 점수가 함께 기록되는 것을 확인했다. 기존 카드가 두 번째 슬롯으로 새는 조건도 상위 3개 취향 기준으로 수정했다.
