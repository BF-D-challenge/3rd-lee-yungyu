## 운영 방식

- Moment 2를 통과한 뒤 플랫폼 주인 우위와 시장 폭 사전 게이트까지 통과한 후보만 포함한다.
- 제품성 점수 내림차순으로 정렬하되, 한 회차에 B2C 6개·1인 B2B 4개를 배치한다.
- 1~5회차는 각 10개, 6회차는 4개, 마지막 7회차는 1인 B2B 2개다.
- 사용자에게 먼저 10개 요약을 보여주고, Yes 후보만 전체 UVP 카드로 하나씩 검증한다.
- 사용자 최종 Yes 전에는 앱 그룹과 54조합을 추가하지 않는다.

## 집계

- 전체: **56개**
- B2C: **30개**
- 1인 사업자 B2B: **26개**
- 사용자 검증 회차: **7회** — 10개 × 5회 + 4개 + 2개
- 사전 제거: **6개** — 플랫폼 주인 우위 Fail 5개 + 시장 폭 Merge 1개
