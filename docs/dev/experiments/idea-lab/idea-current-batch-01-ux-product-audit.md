## 2. 대표 여정

### B09-01 · 첫결과 구조대

- 🟢 이해 — `결제 후 7일·결과물 0개`에서 대상과 위험을 바로 파악
- 🟢 시작 — 결제자 명단을 붙이고 첫 결과물 하나를 정함
- 🟢 완료 — 미제출자 목록과 먼저 보낼 메시지를 받음
- 완전성공: 3/3. 결과가 진도율이 아니라 실제 제출 상태라는 대조가 분명함

### A09-18 · 차지백 제출팩

- 🟢 이해 — 돈을 잃기 전 증거와 마감을 챙긴다는 문제는 선명함
- 🟡 시작 — 어떤 증거가 필요한지는 PG 사유 코드에 의존
- 🔴 완료 — Stripe Smart Disputes와 Chargeflow가 증거 수집·패킷·제출을 이미 자동화
- UX는 2/3이지만 플랫폼 주인 우위로 제품 Fail

### A08-14 · 정비 보증 재방문

- 🟡 이해 — 재방문 기록인지 보증 판정인지 첫 문장에서 갈림
- 🔴 시작 — 정비업체가 가진 보증 조건을 사용자가 직접 입력해야 함
- 🔴 완료 — 무상 처리 가능 여부는 결국 업체에 다시 문의해야 함
- 완전성공: 0/3
