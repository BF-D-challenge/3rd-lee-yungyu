# 아이디어 실제 실험 후보

업데이트: 2026-07-13  
원칙: 합성 UX·제품 게이트를 통과했지만 실제 수요 검증 전인 후보만 둔다. 앱 그룹과 조합에는 아직 추가하지 않는다.

## B09-01 첫결과 구조대

- 상태: **Experiment Pass**
- 타겟: 매달 10~100명 유료 코호트·강의·코칭을 혼자 운영하는 강사
- 유일 주장: 진도율이 아니라 결제 후 7일 안에 첫 결과물을 못 만든 수강생만 찾는다.
- MVP: 결제자 명단 → 첫 결과물 제출 링크 → D+7 미제출자 목록
- 가격 가설: 첫 20명 무료, 다음 코호트 1회 9,900원
- 합성 UX: 3/3
- 실제 통과 조건: 강사 5명 인터뷰 중 3명 문제 인정 + 2명 선결제
- 상세 카드: `idea-user-validation-detail-B09-01.md`
- 감사 보고서: `idea-current-batch-01-ux-product-audit.md`

## B05-01 경력성과증거팩

- 상태: **Experiment Pass**
- 타겟: 이직 준비 중인 20~30대 직장인
- 유일 주장: 이력서 문장마다 면접에서 바로 열 수 있는 원본 파일과 숫자 근거를 연결한다.
- MVP: 이력서 한 줄 → 파일·수치 3개 → 면접용 증거 카드
- 합성 UX: 3/3
- 실제 통과 조건: 구직자 10명 중 3명 유료 지원기간 패스 결제

## C11 부모님공문

- 상태: **Experiment Pass**
- 타겟: 학교·학원 공지가 여러 채널로 오는 다자녀 가정
- 유일 주장: 여러 아이의 공지에서 이번 주 준비물·서명·마감만 가족 한 장으로 합친다.
- MVP: 공지 스크린샷 → 아이별 할 일 → 가족 공유 한 장
- 합성 UX: 3/3
- 실제 통과 조건: 10가구 중 3가구 4주 연속 사용 + 2가구 결제

## LC11 펫시터 인계팩

- 상태: **Experiment Pass**
- 타겟: 카카오톡으로 예약·보고하는 독립 펫시터
- 유일 주장: 보호자 요청부터 식사·산책·배변·특이사항 결과까지 한 링크로 인계한다.
- MVP: 보호자 요청 → 사진·활동 기록 → 확인 링크
- 합성 UX: 3/3
- 실제 통과 조건: 독립 펫시터 5명 중 2명 월 결제
