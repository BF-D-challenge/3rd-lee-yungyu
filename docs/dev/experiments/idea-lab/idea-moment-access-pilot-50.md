# 사용 순간 접근성 파일럿 — 50개

검사일: 2026-07-12  
기준: [아이디어 공통 평가표 v2.1](./idea-evaluation-rubric-v2.md)  
검사 범위: 최근 배치 006~009의 기존 New Pass 52개 중 고정 순서 앞 50개  
앱 데이터 반영: 없음

> 후속 전체 감사에서 나머지 20개까지 검사를 완료했다. 현재 정본은 [사용 순간 접근성 전체 감사 — 70개](./idea-moment-access-audit-70.md)다.

## 결론

- 정확히 **50개**를 `Moment 2/1/0`으로 검사했다.
- **유지 45 / Hold 4 / Fail 1**이다.
- 유지된 45개는 B2C 27개, 1인 사업자 B2B 18개다.
- 걸러진 5개는 모두 B2C다. 공통 문제는 기능이 아니라 **필요한 기록보다 제품 발견이 늦는 것**이다.
- 5개는 폐기하지 않고 [커스텀 아이디어 보관함](./idea-custom-expert-reserve.md)으로 옮겼다. 해당 업계의 고객·예약·접수 채널을 이미 가진 사람에게만 다시 추천한다.
- 이 파일럿에서는 나머지 최근 후보 2개와 과거 복구 후보 18개를 검사하지 않았다.

`유지`는 사용 순간 접근성 게이트만 통과했다는 뜻이다. 실제 결제·사용자 검증까지 완료했다는 뜻은 아니다.

## 유지된 45개가 통과한 이유

- **문제가 보인 뒤 시작해도 늦지 않음:** 보험금 지급대조, 연봉 오퍼 차이, 환율·수수료 대조, 여행 환불 대조처럼 필요한 문서가 이미 있다.
- **예정된 전환 전에 사용자가 검색함:** 항공권 이름 수정, 새 휴대폰 이전, 번호 교체, 휴대폰 삭제는 출발·기기교체·판매 전 준비 기간이 있다.
- **반복 생활·취미 문제임:** 골프 스윙, 자전거 인터벌, 관리비 차이, 사진 백업은 희귀 사건 한 번으로 끝나지 않는다.
- **직업 커뮤니티에서 첫 20명을 찾을 수 있음:** 촬영 백업, 영상 수정큐, PG 명세, 코호트 첫 7일은 사진가·편집자·판매자·강사가 이미 모여 있고 같은 업무를 반복한다.

## 해석

기존 평가에서는 `첫 고객`이 16점 중 2점이라 다른 항목이 높으면 유입 경로가 없어도 Pass가 가능했다. 새 게이트는 이 허점을 막는다.

- `Moment 2`: 제품성 점수대로 사용자 검증 후보 유지
- `Moment 1`: 제품성 점수와 무관하게 Hold
- `Moment 0`: 독립 제품으로 Fail

이번 표본의 제거율은 **10%**다. B2B는 타겟 직업 커뮤니티와 반복 업무가 명확해 18개 모두 유지됐고, B2C는 32개 중 5개가 걸러져 **27개만 유지**됐다.

일반 포트폴리오에서는 제거하지만 아이디어 자체를 삭제하지 않는다. 5개 모두 `Custom Reserve`로 보존하며, 도메인 전문가의 실제 보유 채널이 확인될 때만 재심사한다.
