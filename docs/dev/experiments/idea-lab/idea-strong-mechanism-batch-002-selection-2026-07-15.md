# 강한 메커니즘 배치 002 — gate review 재선별

- 과거 gate review·hard gate pass: 570개
- 앱형 카드 재작성 대상: 5개
- 기준: 제목·범위·처리 설명을 좁히면 입력 1개 → 결과 1개가 남고, 같은 문제 안에서 세 결과 변화를 만들 수 있음
- 앱 반영: 아직 하지 않음

| 순서 | 원본 | 선택 이유 |
|---:|---|---|
| 1 | CarMaster | 실제 월 매출이 있는 차량 청취 전용 음악 마스터링이며 음악 파일 하나가 저장 가능한 오디오 하나로 끝난다. |
| 2 | SQLFlow | CSV 파일 하나를 열 구조에 맞는 SQL 스크립트 하나로 바꾸는 개발자용 세로 조각이 분명하다. |
| 3 | Chordprep | 코드가 포함된 곡 텍스트 하나를 조옮김 계산해 연주용 코드 시트 한 장으로 만드는 흐름이 선명하다. |
| 4 | Mockphine | 엔드포인트 정의 하나를 받아 프론트엔드가 즉시 호출할 로컬 모의 응답 하나를 돌려주는 개발 흐름이다. |
| 5 | Hyperfocal | 참고 이미지 하나에서 편집기에 다시 쓸 색감 프리셋 파일 하나를 만드는 반복 제작 흐름이다. |
