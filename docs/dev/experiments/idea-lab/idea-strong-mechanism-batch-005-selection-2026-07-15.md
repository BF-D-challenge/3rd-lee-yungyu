# 강한 메커니즘 배치 005 — 기존 근접 통과 재활용

- 카드 작성 대상: 10개
- 기준: 과거 Stress·Latin 결과를 재사용해 메커니즘은 살아 있고 카드 범위·제목만 고치면 되는 후보를 우선함
- 제외: 사전 수집 파일 의존, 범용 AI·계산기 대체, 이미 앱에 있는 원본
- 앱 반영: 아직 하지 않음

| 순서 | 원본 | 선택 이유 |
|---:|---|---|
| 1 | Photo String Art | 사진 한 장을 바늘 번호와 선 연결 순서가 있는 스트링 아트 도안 한 장으로 바꾸는 전문 알고리즘 결과가 분명하다. |
| 2 | Smokify | 금연 시작 정보 한 줄로 지금까지의 금연 일수·피우지 않은 담배·절약액을 계산하는 결과가 즉시 보인다. |
| 3 | EXIF Data | 사진 한 장에서 공개 전 숨기고 싶은 위치·기기·전체 EXIF를 제거한 새 파일 하나를 만드는 개인정보 처리 흐름이다. |
| 4 | mermaidonline.live | Mermaid 코드 한 덩어리를 문서에 바로 붙일 수 있는 다이어그램 이미지 한 장으로 렌더링하는 개발자 흐름이다. |
| 5 | SketchCut PRO | 판재와 부품 치수 텍스트 한 덩어리를 자투리가 적은 절단 배치 PDF 한 장으로 계산하는 전문 제작 도구다. |
| 6 | Sketch Clean | 연필 스케치 사진 한 장에서 종이 그림자와 얼룩을 지우고 깨끗한 투명 배경 선화 PNG 하나를 만든다. |
| 7 | iPray: Adhan · Prayer · Qibla | 현재 위치 하나로 오늘의 기도 시각과 키블라 방향을 계산해 낯선 장소에서 바로 확인하게 하는 생활 계산 도구다. |
| 8 | 기타 튜너 - LikeTones | 마이크로 기타 한 줄의 음정을 듣고 목표 음과의 차이를 즉시 보여주는 전문 튜닝 피드백이다. |
| 9 | JSON Tools | JSON 텍스트 하나를 검사해 배포를 막는 문법 오류 위치 하나를 즉시 표시하는 개발자 검증 흐름이다. |
| 10 | Massive Dev Chart Timer | 필름·현상액·희석·온도 텍스트 한 덩어리로 보정된 현상 시간표 하나를 계산하는 아날로그 사진 전문 도구다. |
