# 강한 메커니즘 배치 010 — Chrome 전문 도구 10개

- 카드 작성 대상: 10개
- 기준: 카드 감사·앱 포트폴리오에 한 번도 쓰이지 않은 chrome_web_store Candidate
- 우선순위: 입력 1개 → 결정적 전문 처리 1회 → 파일·보고서·카드 결과 1개
- 제외: 브라우저 기본 기능 대체, 상대방 행동, 상시 감시, 범용 AI 생성
