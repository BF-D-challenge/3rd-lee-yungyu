## 검토 방법

아래 10개는 UX 2/3, 제품성 12점 이상, Moment 2를 통과한 후보다. 각 후보에 대해 `Yes / No / 보류 + 한 줄 이유`로 답한다.

- **Yes:** 전체 UVP 카드로 한 개씩 상세 검증
- **No:** 탈락 이유를 기록하고 다음 후보로 이동
- **보류:** 확인해야 할 가격·채널·입력 조건을 한 가지로 고정

## 1. 연봉 오퍼 차이 — B2C

- **유일 주장:** 구두로 들은 조건과 서면 오퍼에서 달라진 한 줄만 찾는다.
- **타겟:** 최종 합격 뒤 연봉·직급·입사일을 확인하는 이직자.
- **도달 경로:** 오퍼레터를 받은 직후의 이직·연봉협상 커뮤니티와 검색.
- **결제:** 오퍼 비교 1회 3,900원.
- **MVP:** 구두 조건 메모 + 오퍼 PDF → 조건 차이표 → 확인 질문 3개.

## 2. 항공권 이름 수정팩 — B2C

- **유일 주장:** 여권과 예약에서 출발을 막을 수 있는 차이만 출발 전에 찾는다.
- **타겟:** 해외 항공권을 직접 예약하고 영문 이름이 불안한 여행자.
- **도달 경로:** `항공권 이름 여권 다름`, `영문 이름 수정` 고의도 검색.
- **결제:** 예약 1건 2,900원.
- **MVP:** 여권 정보 직접 입력 + 예약 캡처 → 이름·생년월일 차이 → 항공사 문의 체크리스트.

## 3. 2단계 번호 교체표 — B2C

- **유일 주장:** 전화번호를 바꾼 뒤 인증이 막힐 계정만 미리 찾는다.
- **타겟:** 번호 이동이 아니라 실제 전화번호 변경을 앞둔 사용자.
- **도달 경로:** `전화번호 변경 해야 할 일`, `번호 바꾸면 인증` 검색.
- **결제:** 번호교체 7일 패스 3,900원.
- **MVP:** 자주 쓰는 계정 직접 선택 → 이전/새 번호 상태 확인 → 변경 전 남은 계정.

## 4. 방문수리 확정서 — B2C

- **유일 주장:** 기사 방문 전에 증상·기본 견적·추가금 상한을 고객과 기사에게 같은 한 장으로 확인받는다.
- **타겟:** 가전·보일러·에어컨 방문수리를 예약한 가정 사용자.
- **도달 경로:** 방문수리 예약 직후와 `수리 추가금`, `출장비 분쟁` 고의도 검색.
- **결제:** 수리 1건 2,900원.
- **MVP:** 예약 문자·증상 입력 → 견적·추가금 조건 확인 → 기사에게 보낼 확인 링크.

## 5. 보험금 지급내역 대조 — B2C

- **유일 주장:** 보험사의 지급 결정서와 실제 입금에서 빠진 금액만 보여준다.
- **타겟:** 보험금 지급 결정 알림과 입금을 받은 개인 청구자.
- **도달 경로:** `보험금 지급 내역`, `보험금 적게 들어옴` 고의도 검색.
- **결제:** 청구 1건 3,900원.
- **MVP:** 결정서 캡처 + 입금 금액 → 항목별 차이 → 보험사에 물을 질문 목록.

## 6. 사고 렌트비 보상팩 — B2C

- **유일 주장:** 교통사고 대차 영수증과 접수 증거를 보상 담당자가 요구하는 제출 순서로 묶는다.
- **타겟:** 사고 처리 중 렌터카·대차 비용을 직접 청구해야 하는 운전자.
- **도달 경로:** 사고 접수 뒤 `교통사고 렌트비 청구`, `대차료 보상` 검색.
- **결제:** 사고 1건 3,900원.
- **MVP:** 사고 접수·영수증 캡처 → 누락 자료 표시 → 제출용 한 파일.

## 7. 수정횟수 계산표 — 1인 B2B

- **유일 주장:** 계약에 남은 수정과 지금부터 추가금인 요청을 고객과 같은 숫자로 본다.
- **타겟:** 디자인·영상·개발 프로젝트를 혼자 수주하는 프리랜서.
- **도달 경로:** 프리랜서 커뮤니티와 실제 프로젝트의 수정 요청 순간.
- **결제:** 프로젝트 1건 4,900원.
- **MVP:** 수정 조항·요청 메시지 → 포함/초과 분류 → 남은 횟수·추가견적 승인 링크.

## 8. 코호트 첫 7일 — 1인 B2B

- **유일 주장:** 결제한 수강생 중 7일째에도 첫 결과가 없는 사람만 보여준다.
- **타겟:** 10~100명 규모 온라인 강의·챌린지를 직접 운영하는 강사.
- **도달 경로:** 클래스·코호트 운영자 커뮤니티와 새 기수 시작일.
- **결제:** 코호트 1회 9,900원.
- **MVP:** 수강생 명단·첫 결과 제출 링크 → 제출/미제출 상태 → 먼저 연락할 사람 목록.

## 9. 주간 실행보고 — 1인 B2B

- **유일 주장:** 카톡방 50명 중 이번 주 결과가 없는 사람만 운영자에게 보여준다.
- **타겟:** 유료 스터디·챌린지·멤버십을 혼자 운영하는 사람.
- **도달 경로:** 커뮤니티 운영자 모임과 매주 반복되는 마감일.
- **결제:** 월 12,900원.
- **MVP:** 멤버 명단·제출 링크 → 이번 주 증거 제출 → 미제출 목록.

## 10. 착수 조건 한 장 — 1인 B2B

- **유일 주장:** 계약금·자료·킥오프 중 아직 일을 막는 조건 하나만 보여준다.
- **타겟:** 여러 고객 프로젝트를 동시에 시작하는 프리랜서·소형 대행사.
- **도달 경로:** 프리랜서·대행사 커뮤니티와 계약 직후 프로젝트 시작 순간.
- **결제:** 프로젝트 1건 2,900원.
- **MVP:** 착수 조건 3개·고객 확인 링크 → 완료/미완료 상태 → 시작을 막는 한 조건.
