## 판정 방법과 근거 한계

루브릭의 고정 페르소나를 적용했다. 표의 `Y/Y/Y`는 해당 페르소나가 각각 `10초 이해 / 한국 실제 사용 / 결제 이유`를 모두 Yes로 본다는 뜻이다. `UX 합의`는 이 세 Yes를 동시에 받은 페르소나 수다. `하드게이트`는 한국 반복성, 기존 대안 대비 차이, 결제 결과, 4주 MVP를 묶어 판정했다.

해외 실수요 앵커는 지정된 로컬 TrustMRR 원본의 매출 신호다. 예: Practiceme $3.4k/30d, CueCam $4.8k/30d, Sufler $5.2k/30d, SnapLearn $21/30d, Studix $44/30d, Replay It $40/30d, Show Notes Generator $78/30d, VideoToText $902/30d, AI Renamer $143/30d, Risenote $984/30d. 이는 원본 카테고리의 지불 가능성 근거이지 한국 수요의 증명이 아니다. 국내 근거는 기존 저장 자료의 K1~K4를 비교했다.

- K1: KB손해보험 공식 앱 리뷰 — 사진 서류 제출과 보상 청구·처리 현황 사용 행동이 확인됨. [`insurance-claim.json`](../../../../.firecrawl/korea-useful-batch-002-extra/insurance-claim.json)
- K2: YTN 택배 파손 보도 — 파손 시 사진·동영상 증거와 기한 내 통지 행동. [`parcel-damage.json`](../../../../.firecrawl/korea-useful-batch-002-extra/parcel-damage.json)
- K3: 방송미디어통신위원회 블로그 — 통신사 유료 부가서비스를 명세서에서 확인·해지하려는 행동. [`telecom.json`](../../../../.firecrawl/korea-useful-batch-002-extra/telecom.json)
- K4: 국내 이사 공과금 정산 안내 — 계량기·전기·가스·수도 정산을 따로 확인하는 행동. [`moveout.json`](../../../../.firecrawl/korea-useful-batch-002-extra/moveout.json)
- K5: 국내 오답노트 검색 결과 — 기출 PDF를 불러오고 오답을 반복하는 수험 행동. [`01-wrong-note.json`](../../../../.firecrawl/korea-useful-batch-003/01-wrong-note.json)
- K6: 국내 텔레프롬프터 검색 결과 — 혼자 유튜브·강의 촬영을 하며 대본을 보려는 반복 행동. [`teleprompter-kr.json`](../../../../.firecrawl/korea-useful-batch-002/teleprompter-kr.json)
- K7: 국내 자막 검수 검색 결과 — Subtitle Edit로 자막을 최종 검수하는 제작 행동. [`28-short-caption.json`](../../../../.firecrawl/korea-useful-batch-003/28-short-caption.json)

이번 요청에 따라 Firecrawl CLI로 추가 한국 검색을 시도했으나 실행 환경 DNS에서 `api.firecrawl.dev`가 해석되지 않아 `.firecrawl/luna-batch-004-b/` 검색 파일은 생성되지 않았다. 따라서 K1~K4와 로컬 원본만 사용했고, 새 국내 수치가 필요한 후보는 보수적으로 탈락시켰다.

## 최종 통과 후보 브리프

### LB04-01 회독시계

- 타겟: 자기 PDF와 기출 해설로 준비하는 3~6개월차 자격증 수험생.
- 기존 앱 대비 차이: 문제를 생성하거나 요약하지 않고, 사용자가 페이지를 읽고 `다시 볼 날짜`를 직접 확정한다. 페이지별 마지막 회독·체크 결과·다음 복습일을 소유한다.
- 전체 플로우: PDF 1개 업로드 → 페이지를 읽고 `이해/불안` 체크 → 오늘·3일·7일 큐 생성 → 복습 완료/미룸 → 회독 전후 미확신 페이지 비교.
- 가장 작은 MVP: PDF 1개, 페이지 번호·체크 상태·세 번의 복습일, 오늘 큐와 CSV 내보내기.
- 결제 이유: 시험 직전 빠뜨린 페이지를 줄이는 결과에 1회 4,900원 또는 시험 기간 2,900원.

### LB04-02 듣기구간복습

- 타겟: 토익·오픽을 준비하며 자기 강의 음원과 스크립트를 가진 직장인 수험생.
- 기존 앱 대비 차이: 전체 음성 대화나 단어 생성을 하지 않고, 사용자가 막힌 구간을 20초 단위로 표시하면 그 구간의 복습 이력과 다음 재생일을 관리한다.
- 전체 플로우: 음원·스크립트 추가 → 막힌 구간 표시 → 구간 반복 → 들은 뒤 이해 여부 체크 → 다음 복습일 → 첫 기록과 최근 기록의 이해도 비교.
- 가장 작은 MVP: 오디오 1개, 구간 시작·끝 수동 지정, 3단계 복습 큐, 구간별 전후 상태와 M4A/CSV 내보내기.
- 결제 이유: 시험일까지 안 들리는 구간을 줄이는 4주 이용권 5,900원.

### LB04-03 실기루틴판

- 타겟: 미용·조리·제과 등 국가자격 실기에서 같은 동작을 반복하는 수험생.
- 기존 앱 대비 차이: 동작을 판정하거나 정답 영상을 대량 공급하지 않고, 수험생이 정한 체크포인트와 전후 영상을 한 시험 루틴 상태로 묶는다.
- 전체 플로우: 루틴 5개 입력 → 한 루틴 촬영 → 체크포인트 완료·메모 → 전 영상과 이번 영상 나란히 비교 → 다음 연습일과 미완료 하나 표시.
- 가장 작은 MVP: 루틴 하나, 영상 두 개, 체크포인트 5개, 전후 비교와 MP4/리포트 내보내기.
- 결제 이유: 학원 피드백 없는 연습 주간에 합격 전까지 쓸 수 있는 9,900원 단건 구매.

### LB04-07 자막용어검수

- 타겟: 주 2회 이상 한국어 강의·쇼츠를 만드는 1인 크리에이터와 강사.
- 기존 앱 대비 차이: 자막을 만들지 않고, 본인이 지정한 표기집과 이전 SRT를 기준으로 용어·숫자·고유명사의 회차 간 불일치를 검수한다.
- 전체 플로우: 기준 용어표 입력 → SRT 여러 개 업로드 → 불일치 위치 표시 → 수정 승인/보류 → 수정 SRT와 검수 리포트 내보내기.
- 가장 작은 MVP: SRT 3개, 용어표 CSV, 불일치 목록·일괄 치환 확인·SRT 다운로드.
- 결제 이유: 매주 재사용하는 검수 시간을 줄이는 월 4,900원 또는 100개 파일 9,900원.

### LB04-08 강의자료변경표

- 타겟: 매 기수 슬라이드를 고쳐 쓰는 온라인 강사·사내교육 발표자.
- 기존 앱 대비 차이: 새 강의자료를 생성하지 않고, 지난 PDF와 이번 PDF의 장표·발표자 노트 변경을 소유해 발표 전 누락 확인을 강제한다.
- 전체 플로우: 이전·이번 PDF 업로드 → 장표 단위 차이 표시 → 변경 사유·발표 멘트 확인 → 미확인 장표 큐 → 변경표 PDF 내보내기.
- 가장 작은 MVP: PDF 2개, 페이지 이미지 비교, 변경 장표 목록, 확인 상태와 PDF 내보내기.
- 결제 이유: 기수마다 배포 사고를 막는 강사 단건 결제 7,900원 또는 월 5,900원.

### LB04-11 댓글약속장부

- 타겟: 주 1회 이상 콘텐츠를 올리고 댓글에서 후속 주제를 약속하는 1인 크리에이터.
- 기존 앱 대비 차이: 댓글을 요약하거나 다음 대본을 만들지 않고, 약속한 주제의 제작·게시 상태와 원댓글을 끝까지 연결한다.
- 전체 플로우: 댓글 캡처/텍스트 추가 → 약속 문장 확인 → 마감일·상태 지정 → 다음 촬영 큐 → 게시 링크 기록 → 원댓글과 완료 결과 비교.
- 가장 작은 MVP: 수동 댓글 붙여넣기, 약속 20개, 상태 4단계, 마감 알림과 CSV/공개 링크 내보내기.
- 결제 이유: 팬에게 한 약속을 놓쳐 신뢰를 잃는 비용을 줄이는 월 3,900원.

### LB04-13 파일버전찾기

- 타겟: 카톡·메일로 기획서와 계약 초안을 주고받는 프리랜서·소규모 사업자.
- 기존 앱 대비 차이: PDF와 문서를 대화형으로 요약하지 않고, 두 파일의 변경 문장·추가 조항·삭제 조항을 보여주며 사용자가 최종본을 선택하고 비교 리포트를 보관한다.
- 전체 플로우: 파일 두 개 선택 → 텍스트·페이지 차이 표시 → 변경 항목 승인/메모 → 최종본 지정 → 비교 PDF와 원본 묶음 내보내기.
- 가장 작은 MVP: PDF/DOCX 2개, 텍스트 diff, 페이지 위치, 최종본 별표, ZIP 내보내기.
- 결제 이유: 잘못된 버전 발송·재작업을 막는 건당 2,900원 또는 월 4,900원.

### LB04-14 사진전후대조

- 타겟: 수리·청소·시공 결과를 고객에게 사진으로 증명하는 1인 작업자.
- 기존 앱 대비 차이: 사진 보정이나 앨범 정리가 아니라, 같은 위치의 전후 사진을 맞춰 변화 영역·날짜·작업 메모를 한 장의 납품 증거로 만든다.
- 전체 플로우: 작업 전 사진 등록 → 작업 후 사진 촬영 가이드 → 같은 구도 맞춤 → 전후 슬라이더·메모 확인 → 고객용 한 장/PDF 전송.
- 가장 작은 MVP: 전·후 2장, 수동 정렬, 날짜·메모·로고 없는 PDF/PNG 내보내기.
- 결제 이유: 건별 결과 보고서를 고객에게 보내는 순간 1건 1,900원 또는 월 5,900원.

### LB04-17 스크린샷약속

- 타겟: 카톡으로 외주·공구·행사 일을 조율하며 개인적으로 마감 약속을 관리하는 1인 운영자.
- 기존 앱 대비 차이: 캡처를 일정으로 옮기는 데서 끝나지 않고, 원문 위치·담당자·마감·증거 파일·완료 상태를 한 레코드로 묶어 미완료를 계속 소유한다.
- 전체 플로우: 카톡 캡처 추가 → 약속 문장·날짜 수동 확인 → 상태 지정 → 마감 전 큐 → 완료 증거 첨부 → 약속 원문과 완료 기록 내보내기.
- 가장 작은 MVP: 이미지 1장, 약속 1~5개 수동 확인, 상태 4단계, 마감 알림과 CSV/PDF 내보내기.
- 결제 이유: 납기 누락으로 생기는 재촉·환불·신뢰 손실을 줄이는 월 3,900원.

## 실패 후보 한 문장 사유

- LB04-04: 오답 단어장·간격 복습은 이미 수험 앱과 스프레드시트가 제공하며 P2/P3가 별도 결제 결과를 못 봤다.
- LB04-05: 강의 진도는 국내 강의 플랫폼이 로그인 안에서 이미 소유한다.
- LB04-06: 점수 추세는 학원·시험 플랫폼의 성적표와 겹치고, 개인이 추가로 사는 결과가 약하다.
- LB04-09: 촬영 테이크의 끊김 비교는 `내말속도`·`말버릇컷`의 전후 검수와 중복된다.
- LB04-10: 인쇄 미리보기와 PDF 뷰어로 핵심 결과를 대체할 수 있다.
- LB04-12: 게시 체크리스트는 기본 메모·플랫폼 예약 기능으로 충분하고 해외 매출 앵커도 $0였다.
- LB04-15: 보증 확인은 구매·수리 빈도가 낮고 제조사·쇼핑몰 계정이 이미 원문을 가진다.
- LB04-16: 취소·연기 일정은 캘린더의 수정·알림으로 해결되어 별도 앱의 결제 결과가 없다.
- LB04-18: 사진 백업 여부는 iCloud·Google Photos 등 기본 클라우드가 핵심 결과를 해결한다.
- LB04-19: 파일명·첨부 점검은 탐색기와 제출 사이트의 최종 확인으로 대체되며 월 반복 결제가 약하다.
- LB04-20: 발표 질문 기록은 UX는 통과했지만 숙련 발표자에게 문서·노션 대비 전환 이득이 부족하다.
