# Luna 배치 004 — 비 Firecrawl 웹 재검사 최종판

검사일: 2026-07-12

## 결론

- 재검사 대상: Luna A/B/C가 발굴한 60개 중 1차 통합 통과 20개
- 최종 통과: **6개**
- 보류: **2개**
- 최종 탈락: **12개**
- UX 2/3 미만 후보를 되살린 사례: **0개**
- 현재 실제 앱 조합 수: **1,728개**. 이번 6개는 아직 앱 데이터에 추가하지 않았다.

격리된 Luna 재검색 보고서의 `0/60`은 아이디어 품질 판정이 아니라 DNS 수집 실패 결과다. 따라서 그 숫자는 폐기하고, 루트 환경에서 Firecrawl 없이 일반 웹 검색과 공식 페이지를 직접 확인해 1차 통과 20개를 다시 판정했다.

## 적용 규칙

1. UX 2/3 미만은 즉시 실패한다.
2. UX 2/3 이상이어도 국내 강한 대안, 불명확한 결제 결과, 4주 MVP 초과 중 하나면 탈락한다.
3. 웹 근거가 없다는 이유만으로 자동 탈락시키지 않고, 수집 실패와 시장 부재를 구분한다.
4. 이번 통과 6개도 바로 54조합으로 늘리지 않는다. 대표 카드와 결제 실험을 먼저 통과시킨 뒤 그룹으로 확장한다.
