# Luna 배치 005 D — 쇼핑·여행·지역생활·디지털 생활 25개

`docs/dev/experiments/idea-lab/luna-prompts/batch-005-shared-rules.md`를 먼저 읽고 모든 규칙을 적용한다.

## 담당 영역

- 온라인 쇼핑·배송·반품·가격/구독 손실
- 국내외 여행 준비·예약·현장 마찰
- 동네 서비스·예약·수리·세탁·중고거래
- 개인정보·계정·사진·파일 등 디지털 생활

기존 `숨은구독`, `통신비구멍`, `중고인수증`, `공구입금`, `환불왔나`, `예약금마감`, `언박싱증거`, `공동구매검수`, `스크린샷약속`, `30칸`과 입력·결과가 겹치면 탈락한다. 한국에서 플랫폼이 이미 처리하는 배송조회·기본 환불·가격비교를 다시 만들지 않는다.

## 실행

- 후보를 정확히 25개 만든다.
- Firecrawl을 사용하지 않는다.
- 최종 통과 가능 후보는 공식 페이지·앱스토어·일반 웹 검색으로 국내 대안을 확인한다.
- 결과는 오직 `docs/dev/experiments/idea-lab/luna-batch-005-d-shopping-travel-digital.md`에 쓴다.
- 앱 데이터와 다른 보고서는 수정하지 않는다.
