# 후보 6개 UX 합성 평가 입력

평가자는 아래 카드만 보고 세 과업을 수행한다.

1. 10초 안에 누구의 어떤 문제인지 말한다.
2. 첫 입력 또는 첫 행동을 말한다.
3. 완료 뒤 받는 결과를 말한다.

## V1. 첫 제출 없는 수강생 찾기

```text
┌────────────────────────────────────────┐
│ 첫 제출 없는 수강생 찾기             │
│ 유료 코호트 운영자용 · 결제 D+7      │
│                                        │
│ 결제자 이메일 명단 + 여기서 만든      │
│ 첫 결과 제출 링크                     │
│                ↓                       │
│ 제출 기록 없는 이름 + 복사할 확인문자 │
│                                        │
│ [명단 붙여넣기]                        │
└────────────────────────────────────────┘
```

서비스 한 줄: 유료 코호트 운영자가 진도율 대신 첫 결과물을 못 낸 사람을 먼저 찾는 도구.

## V2. 12명 3테이블 자리표

```text
┌────────────────────────────────────────┐
│ 12명 3테이블 자리표                   │
│ 행사 운영자용 · 자리 충돌 없이        │
│                                        │
│ 참석자 이름 + 테이블 3개 +            │
│ 함께/따로 조건 1개                    │
│                ↓                       │
│ 조건을 지킨 테이블별 이름 + 충돌 0건  │
│                                        │
│ [참석자 이름 붙여넣기]                 │
└────────────────────────────────────────┘
```

서비스 한 줄: 작은 행사 참석자와 관계 조건을 넣어 세 테이블 자리표를 만드는 도구.

## V3. 이번 주 준비물 한 장

```text
┌────────────────────────────────────────┐
│ 이번 주 준비물 한 장                  │
│ 다자녀 부모용 · 공지 5장을 다시       │
│ 읽지 않게                              │
│                                        │
│ 공지 사진을 1장씩 추가 · 최대 5장     │
│                ↓                       │
│ 아이별 준비물·서명·마감 + 원문 한 줄  │
│                                        │
│ [공지 사진 올리기]                     │
└────────────────────────────────────────┘
```

서비스 한 줄: 여러 아이 공지에서 앞으로 7일의 준비물·서명·마감만 뽑는 가족 도구.

## V4. 게임형 인적성 5분 리허설

```text
┌────────────────────────────────────────┐
│ 게임형 인적성 5분 리허설              │
│ 게임형 평가 초대를 받은 지원자용      │
│                                        │
│ 평가 유형 1개 선택 → 5분 연습         │
│                ↓                       │
│ 오래 걸린 문제 3개 · 걸린 초 · 정답   │
│ + 다음 연습 기준 1줄                   │
│                                        │
│ [낯선 유형 고르기]                     │
└────────────────────────────────────────┘
```

서비스 한 줄: 게임형 채용 평가 전에 유형 하나를 짧게 연습하고 느린 판단을 복기하는 도구.

## V5. 영상 중간 확인 한 문제

```text
┌────────────────────────────────────────┐
│ 영상 중간 확인 한 문제                │
│ 끝까지 봤음 대신 핵심을 이해했는지    │
│                                        │
│ 영상 URL + 질문을 넣을 시점 + 한 문제 │
│                ↓                       │
│ 답해야 이어 보는 링크 + 정답/미응답   │
│                                        │
│ [영상 URL 넣기]                        │
└────────────────────────────────────────┘
```

서비스 한 줄: 교육 영상 한 곳에 질문 하나를 넣어 시청 완료가 아닌 이해 증거를 받는 도구.

## V6. 60초 한 작업 매뉴얼

```text
┌────────────────────────────────────────┐
│ 60초 한 작업 매뉴얼                   │
│ 같은 클릭 순서를 세 번 설명하지 않게 │
│                                        │
│ 화면 녹화 1개                         │
│                ↓                       │
│ 단계 3줄 + 원본 영상의 해당 초 링크   │
│                                        │
│ [60초 녹화 올리기]                    │
└────────────────────────────────────────┘
```

서비스 한 줄: 실제 화면 작업 60초를 다른 사람이 따라 할 단계 세 줄로 바꾸는 도구.
