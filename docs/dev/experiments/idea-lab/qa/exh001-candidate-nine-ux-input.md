# EXH-001 후보 9개 UX 합성 평가 입력

평가자는 아래 ASCII 카드만 보고 각 후보에서 세 과업을 수행한다.

1. 10초 안에 누구의 어떤 문제인지 말한다.
2. 첫 입력 또는 첫 행동을 말한다.
3. 완료 뒤 받는 결과를 말한다.

## V1. AI 답변 3개의 브랜드 언급 비교표

```text
┌────────────────────────────────────────────┐
│ AI 답변 3개의 브랜드 언급 비교표          │
│ AI 추천에서 자사만 빠진 브랜드 마케터용   │
│                                            │
│ 브랜드명 · 대표 질문 1개 · AI 답변 3개   │
│                    ↓                       │
│ 자사/경쟁사 언급 · 출처 URL · 원문 근거   │
│                                            │
│ [답변 붙여넣고 비교]                       │
└────────────────────────────────────────────┘
```

서비스 한 줄: 같은 질문의 AI 답변에서 자사·경쟁사·인용 출처를 근거 문장과 함께 비교하는 도구.

## V2. 명함 한 장을 바로 저장하는 연락처

```text
┌────────────────────────────────────────────┐
│ 명함 한 장을 바로 저장하는 연락처         │

│                    ↓                       │

│ + 휴대폰 연락처 파일(vCard) · 저장 전 수정│
```

서비스 한 줄: 명함 사진을 연락처 필드로 나누어 확인하고 바로 저장 가능한 파일로 만드는 도구.

## V3. 홈페이지에서 뽑은 1쪽 브랜드 카드

```text
┌────────────────────────────────────────────┐
```
