# EXH-001 후보 9개 카드 UX 합성 평가 — 2026-07-14

이 평가는 실제 사용자 검증이 아니라 카드 문구를 빠르게 읽었을 때의 사용성 가설이다. 신입·중급·숙련 세 평가자는 서로의 결과를 보지 않고 `exh001-candidate-nine-ux-input.md`의 ASCII 카드만 봤다.

## 여정

- 신입 김서연: 9개 모두 문제와 결과를 이해했다. `vCard`, `기다릴 날짜`, `Apps Script`, `규칙 1묶음`에서 잠깐 멈췄다.
- 중급 박지민: 9개 모두 통과했다. OCR·사이트 추출의 수정 가능 여부, 이메일 날짜의 의미, 시트 권한, 플랫폼 규칙 최신성을 안심 신호로 요구했다.
- 숙련 이도현: 9개 모두 통과했다. 단건 입력의 효율 천장과 결과 뒤의 배칭·내보내기·재검사 흐름을 공통 한계로 지적했다.

## 결론과 다음 확인

- 카드 이해도 기준으로 9개 모두 통과했다.
- 감정·결제 의향은 합성 평가로 검증되지 않았다.
- 실제 사용자에게는 `단건 결과만으로 결제할지`, `민감한 이메일·명함·영상을 올릴지`, `Apps Script 배포를 끝낼 수 있을지`를 확인해야 한다.
