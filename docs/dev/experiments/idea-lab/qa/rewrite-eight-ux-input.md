# 기존 카드 재작성 8개 UX 입력

각 후보는 제목·설명·첫 화면만 보고 10초 안에 문제, 첫 입력, 완료 결과가 보이는지 평가한다.

## 1. 첫 연락 근거 3줄

회사 공식 URL 하나를 넣으면 소개·최근 변화·내 제안과 이어지는 공개 사실 세 줄과 각각의 출처·확인일을 보여준다. 첫 입력은 `회사 URL 붙여넣기`, 결과는 `공개 근거 3줄 카드`다.

## 2. 1분 음성 3칸 기록

브라우저에서 1분 말하면 `기록 / 다음 행동 / 확인할 것` 세 칸과 담당자·기한, 근거 음성 10초를 보여준다. 첫 입력은 `1분 녹음`, 결과는 `실행 가능한 세 칸 메모`다.

## 3. 앞뒤 두 장 티셔츠 목업

투명 PNG 하나와 인쇄 면(앞/뒤), 티셔츠 색을 고르면 앞·뒤 2D 목업, 인쇄 영역, 색상 2안 비교를 보여준다. 첫 입력은 `PNG+인쇄 면+색`, 결과는 `주문 전 확인용 2D 목업`이다.

## 4. 글 한 편이 30초 무음 숏폼으로

600자 이하 글 또는 공개 URL 하나를 넣으면 세로 장면 네 개와 자막 콘티를 확인하고 30초 무음 MP4 초안을 만든다. 첫 입력은 `글/URL`, 결과는 `장면 4개 무음 숏폼`이다.

## 5. Reddit 질문에 경험 답변 3문장

발견한 Reddit 질문 텍스트와 내가 실제로 해본 해결 한 줄을 넣으면 경험·도움·선택적 상품 연결 세 문장에 `직접 경험/출처 필요` 표시를 준다. 첫 입력은 `질문+내 경험`, 결과는 `광고 티를 줄인 답변 3문장`이다.

## 6. 30초 한 언어 더빙 미리보기

권한 있는 영상 파일을 올린 뒤 30초 구간과 언어 하나를 고르면 고유명사 최대 세 개를 확인하고 번역 자막과 중립 음성을 미리 듣는다. 첫 입력은 `영상 파일+30초 구간+언어`, 결과는 `중립 음성 번역 미리보기`다.

## 7. 같은 식사 행동 7일 체크

물 한 컵·채소 먼저·야식 안 먹기 중 행동 하나를 직접 고르고 오늘 완료만 눌러 7일 동안 같은 행동을 확인한다. 첫 입력은 `행동 하나 선택`, 결과는 `점수·칼로리 없는 7칸 체크표`다.

## 8. 경쟁 URL로 만드는 메인 이미지 1장

Amazon 경쟁 상품 상세 URL 하나와 내 상품 사진 하나를 넣으면 배경·제품 비율·여백 기준 세 줄과 흰 배경 메인 이미지 초안 한 장을 준다. 첫 입력은 `상세 URL+사진`, 결과는 `모바일 썸네일로 비교할 메인 초안 1장`이다.

## 공통 범위

- 로그인·결제·플랫폼 계정 연결·자동 게시·상대방 승인 없음
- 텍스트·URL·파일·사진·브라우저 마이크만 사용
- 입력 1묶음 → 처리 1회 → 결과 1개, 1~3화면
