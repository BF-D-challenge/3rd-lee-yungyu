# `screenshot-thirty-queue` 교체 후보 UX 입력

## 검증 대상

원본: Site Audit Pro  
후보 제목: **공사 전후 사진 인계서**  
한 줄 설명: **현장 사진을 다시 설명하지 말고, 중요한 완료 3건과 전후 차이를 고객이 바로 확인할 한 장으로 보냅니다.**

```text
┌──────────────────────────────────────┐
│ 공사 전후 사진 인계서                │
│ 현장을 떠나기 전에 완료 증거를       │
│ 고객이 알아볼 한 장으로 만드세요.    │
│                                      │
│ 현장명  [ 성수동 카페 2층           ]│
│                                      │
│ ① 수리 전 사진   [ 사진 올리기 ]     │
│ ② 수리 후 사진   [ 사진 올리기 ]     │
│ 위치·작업         [ 주방 싱크 누수 ] │
│ 표시              [ 원 ] [ 화살표 ]  │
│ [ + 완료 항목 추가 (최대 3개) ]      │
│                                      │
│ [ 전후 한 장 만들기 ]                │
└──────────────────────────────────────┘

결과
┌──────────────────────────────────────┐
│ 성수동 카페 2층 · 완료 3건           │
│                                      │
│ 주방 싱크 누수                       │
│ [수리 전]          [수리 후 ✓]       │
│ 누수 연결부 교체 · 7/13 16:20        │
│                                      │
│ [공유 링크 복사] [1쪽 PDF 저장]      │
└──────────────────────────────────────┘
```

## 과업

수리·설치 사진을 이미 가진 사용자가 10초 안에 이 도구가 주는 결과와 첫 입력을 이해하고, 고객에게 전후 증거 한 장을 보내는 경로를 찾을 수 있는지 평가한다.

## 범위

- 사진 1~3쌍, 위치·작업 한 줄, 원/화살표 한 개
- 로그인·결제·업체 배정·고객 승인·메신저 자동 전송 없음
- 결과는 공유 링크 또는 1쪽 PDF
