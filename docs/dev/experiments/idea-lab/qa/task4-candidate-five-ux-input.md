# 4단계 후보 5개 UX 합성 평가 입력

평가자는 아래 ASCII 카드만 보고 각 후보에서 세 과업을 수행한다.

1. 10초 안에 누구의 어떤 문제인지 말한다.
2. 첫 입력 또는 첫 행동을 말한다.
3. 완료 뒤 받는 결과를 말한다.

## V1. 나온 초와 지도 링크가 붙은 여행 장소 목록

```text
┌────────────────────────────────────────────┐
│ 나온 초와 지도 링크가 붙은 여행 장소 목록│
│ 영상 속 장소를 다시 찾는 여행 기획자용   │
│                                            │
│ 자막 있는 공개 YouTube 여행 영상 URL 1개 │
│                    ↓                       │
│ 장소 최대 10곳 · 나온 초 · 근거 자막      │
│ + 지도 링크 · 틀린 장소 제외              │
│                                            │
│ [YouTube URL 넣기]                         │
└────────────────────────────────────────────┘
```

서비스 한 줄: 여행 영상 한 편에서 장소와 나온 시점을 뽑아 검수 가능한 지도 목록으로 만드는 도구.

## V2. 공개 전 SEO 기본 4개 검사

```text
┌────────────────────────────────────────────┐
│ 공개 전 SEO 기본 4개 검사                 │
│ 랜딩페이지를 납품·운영하는 실무자용       │
│                                            │
│ 현재 열어둔 웹페이지                      │
│                    ↓                       │
│ 검색 제목 · 설명 · 대표 제목(H1)          │
│ · 대표 주소(canonical)                    │
│ 있음/없음 · 길이 경고 · HTML 위치         │
│                                            │
│ [현재 페이지 검사]                        │
└────────────────────────────────────────────┘
```

서비스 한 줄: 공개 직전 페이지에서 검색 노출 기본 네 항목의 누락과 위치를 확인하는 브라우저 도구.

## V3. 구글 검색 확장 정보 오류, 고친 코드와 비교

```text
┌────────────────────────────────────────────┐
│ 구글 검색 확장 정보 오류, 고친 코드와 비교│
│ 리치 결과 오류를 고치는 SEO·웹 실무자용   │
│                                            │
│ 현재 페이지 검사 → 오류 블록 1개 선택     │
│                    ↓                       │
│ JSON-LD(검색 확장 정보를 만드는 코드)     │
│ 원본 / 수정 제안 / 바뀐 줄 / 재검사       │
│ 자동 반영 안 함                           │
│                                            │
│ [현재 페이지 구조화 데이터 검사]          │
└────────────────────────────────────────────┘
```

서비스 한 줄: 구조화 데이터 오류 코드를 실제 JSON 경로와 검토할 수정 블록으로 바꾸는 브라우저 도구.

## V4. 세 칸만 눌러 만드는 웹 표

```text
┌────────────────────────────────────────────┐
│ 세 칸만 눌러 만드는 웹 표                 │
│ 공개 목록을 반복해서 엑셀로 옮기는 사람용 │
│                                            │
│ 서로 다른 열의 예시 셀 1~3개              │
│                    ↓                       │
│ 같은 구조의 행·열 미리보기 + 되돌리기     │
│ + CSV                                      │
│                                            │
│ [현재 페이지에서 셀 고르기]                │
└────────────────────────────────────────────┘
```

서비스 한 줄: CSS 선택자 대신 예시 셀을 눌러 현재 공개 페이지의 반복 목록을 CSV로 만드는 확장.

서비스 한 줄: 긴 WCAG 보고서 대신 현재 페이지에서 먼저 고칠 사용자 차단 오류 다섯 개를 보여주는 확장.
