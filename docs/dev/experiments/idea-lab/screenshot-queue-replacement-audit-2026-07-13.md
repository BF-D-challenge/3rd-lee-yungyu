## 1. 교체 이유

기존 카드는 문제 전에 네이티브 사진 권한과 설치가 필요하고, 저장한 스크린샷을 30개 큐로 다시 보는 이득이 기본 사진 앱보다 명확하지 않았다. 통합 8,406개 원본 장부에도 `mymind Serendipity`가 없어 이름만으로 계보를 만들 수 없었다.

Site Audit Pro는 현장 이슈에 사진·설명·담당자를 붙이고 주석이 있는 PDF/CSV 보고서를 공유하는 유료 제품이다. 통합 장부에는 App Store 원본 `430234732`로 존재하며, 미국 공식 페이지에서 4.8점·평가 1.1만개·$12.90 결제와 작동 흐름을 확인했다.

- 장부 원본: `https://apps.apple.com/au/app/site-audit-pro/id430234732`
- 기능 확인: `https://apps.apple.com/us/app/site-audit-pro/id430234732`
- 공식 제품 페이지: `https://siteauditpro.com/`

## 2. 카드 풀

### 결제자

1. 한 달에 2~5곳을 마감하는 소형 인테리어 시공자
2. 여러 매장의 소규모 수리를 맡는 시설관리 담당자
3. 간판·에어컨·가구 설치를 직접 인계하는 1인 기사

### 필요한 순간

1. 수리·설치 완료 점검을 끝내고 현장 사진이 쌓인 직후
2. 고객이 완료된 부분의 사진 증거를 다시 요청한 순간
3. 잔금·월 정산을 청구하기 한 시간 전

### 한 끗

1. 프로젝트 관리 제거 → `사진 3장 공사 완료서`
2. 이슈 목록을 같은 위치 전·후 사진으로 교체 → `수리 전후 한 장 보고서`
3. 사진에 원·화살표 하나 추가 → `표시가 붙은 현장 사진 PDF`

## 3. 27조합 감사

각 칸은 `문제 발생 후 시작 / 현실적 입력 / 즉시 소유 결과 / 반나절~2일 웹 MVP` 네 조건 통과를 뜻한다.

| 결제자 \ 순간 | 완료 점검 직후 | 고객의 증거 요청 | 잔금·정산 직전 |
|---|:---:|:---:|:---:|
| 소형 인테리어 시공자 | ✓✓✓ | ✓✓✓ | ✓✓✓ |
| 매장 시설관리 담당자 | ✓✓✓ | ✓✓✓ | ✓✓✓ |
| 1인 설치 기사 | ✓✓✓ | ✓✓✓ | ✓✓✓ |

세 한 끗 모두 위 9칸에 적용 가능하므로 27/27 통과다. 입력은 사용자가 이미 가진 사진 1~3쌍과 설명 한 줄이고, 결과는 공유 링크 또는 1쪽 PDF다. 고객의 새 승인이나 시공 업체 API는 필요 없다.
