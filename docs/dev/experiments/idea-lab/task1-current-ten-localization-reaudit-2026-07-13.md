|---|---:|---|
| 현재 카드 그대로 유지 | 0 | 없음 |
| 한국형 카드 재작성 | 8 | `cross-post-relay`, `connected-knowledge`, `class-material-coach`, `track-form-coach`, `small-business-site`, `safe-message-reply`, `bulk-shop-listing`, `social-comment-guard` |

## UX 합성 평가

신입·중급·숙련 세 평가자가 생존 카드 12개(재작성 8개+후보 통과 4개)의 문제·첫 입력·완료 결과를 독립적으로 설명했다.

- 신입: 12/12 PASS
- 중급: 12/12 PASS
- 숙련: 12/12 PASS
- 최종: 모든 카드 **3/3**

공통 개선점은 기능 실패가 아니라 입력·판정 용어였다. 앱 반영 전 문구에서 다음을 풀어 쓴다.

1. `CSV·XLSX`는 “카드사에서 받은 거래 파일” 예시를 함께 표시
2. `SVG`는 “글자와 상자를 직접 고칠 수 있는 그림 파일”로 표시
3. 개인정보는 **업로드 전에** 가린다고 순서를 명시
4. `7일 안 본 탭`, `위험 댓글`, `가격 인상`의 판정 기준을 결과 화면에 표시
