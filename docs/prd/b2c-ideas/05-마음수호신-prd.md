## 처음 보는 사람을 위한 한 문장

**마음수호신**은 가벼운 사주·MBTI를 즐기는 성인이 일상적인 걱정 한 줄을 자기만의 작은 수호동물에게 맡기고, 60~90초의 짧은 의식으로 마음을 내려놓게 돕는 모바일 퍼스트 웹 앱이다.

사주는 사용자를 데려오는 익숙한 자기이해 언어이자 수호동물의 테마일 뿐이다. 앱은 미래를 예언하거나 불안장애를 진단·치료하지 않는다.

## 0-A. 포지셔닝 — UVP·타겟·차별점·플로우

### UVP (통증 → 대조 훅)

> **새벽 1시, 답장 없는 채팅방을 열 번 다시 연 당신은 누구에게 이 걱정을 맡길 건가요?**
> 지금까지: AI에게 설명을 더 쓰거나 운세를 다시 보고, 명상 목록에서 무엇을 틀지 고르다 20분이 지났습니다.
> 이제부터: 걱정 한 줄을 작은 수호동물에게 맡기고 90초 의식 하나를 끝내면, 화면을 닫고 원래 하던 일로 돌아갑니다.

Finch는 펫 애착과 셀프케어의 반복을 ARR 약 3,000만 달러로 증명했고, 포스텔러는 국내 월 140만 사용자의 단건 결제를 증명했다. 마음수호신은 더 오래 붙잡는 채팅이나 예언 대신 **판단 없이 맡기고 끝내는 90초**를 판다.

### 타겟

**사주·MBTI·타로를 가벼운 자기이해로 즐기며, 답장·업무 피드백 뒤 일상적 불안을 반복하지만 상담이나 지인 연락은 부담스러운 만 19세 이상 사용자.** 임상 위기·진단·치료·예언을 원하는 사람은 대상이 아니다.

### 기존에 잘되는 것 vs 차별점

| 잘되는 것 | 잘하는 것 | 빈틈 |
|---|---|---|
| **Finch** (ARR 약 $30M) | 펫 애착을 셀프케어 반복과 결제로 연결 | 장기 육성·스트릭·꾸미기 중심이라 지금의 걱정을 짧게 맡기고 종료하는 데 초점이 없음 |
| **포스텔러** (월 140만 사용자) | 익숙한 사주 입구와 소액 단건 결제 | 결과를 읽는 경험이라 오늘의 감정과 행동을 직접 마치게 하지는 않음 |
| **일반 AI·명상 앱** | 즉시 답변과 많은 콘텐츠 | 무한 대화가 이어지거나, 불안한 순간에 다시 콘텐츠를 골라야 함 |

유일 주장 하나로 압축: **“일상 불안을 혼자 반복 확인하는 성인을 위한, 걱정 한 줄을 수호동물에게 맡기고 90초 안에 대화를 끝내는 유일한 비예언 위로 의식.”**

### 전체 플로우 (5단계)

**사주 테마로 수호동물 만나기 → 걱정 한 줄·강도 입력 → 감정 이름과 90초 의식 수행 → 걱정을 내려놓고 강도 다시 기록 → 원문 없이 정원에 행동 흔적 저장**

## 1. 문제 정의

### 표면 문제

불안하거나 마음이 복잡한 순간에 누군가에게 말하고 싶지만, 상담은 무겁고 지인에게 털어놓기는 부담스럽다.

### 본질적 문제

사용자가 원하는 것은 긴 분석이나 정답이 아니라 **지금 이 감정을 판단 없이 잠시 맡아줄 대상과, 생각을 멈추게 해줄 짧고 안전한 종료 의식**이다.

일반 AI 채팅은 즉시 답하지만 사용자가 계속 질문하게 만들 수 있다. 정적 운세·심리 카드는 읽고 끝나 관계와 행동이 남지 않는다. 불안한 순간에는 설명을 더 요구하는 도구보다, 떠나도 상처받지 않고 돌아오면 반겨주는 존재가 필요하다.

### 현재 사람들이 푸는 방식

- ChatGPT나 AI 상담 앱에 고민을 길게 입력한다.
- 사주·운세·타로 앱에서 미래에 대한 확신을 구한다.
- MBTI·심리테스트 결과를 읽으며 자신을 설명한다.
- 명상·호흡 앱의 콘텐츠를 찾아 재생한다.
- 친구에게 메시지를 쓰거나 혼자 참는다.

### 기존 방식의 빈틈

- 무한 채팅은 종료점이 없어 확인 강박과 반추를 늘릴 수 있다.
- 운세는 가볍지만 미래를 단정하거나 불안을 과금으로 자극할 위험이 있다.
- 명상 라이브러리는 불안한 순간에 무엇을 골라야 하는지 다시 결정해야 한다.
- 펫 앱은 강한 반복성을 보이지만, 먹이·출석·상태 악화가 취약한 사용자에게 죄책감이 될 수 있다.

### Why now?

한국 Google Trends의 최근 12개월 상대 관심도에서 `사주`는 같은 비교 묶음의 `MBTI·꿈해몽·궁합·별자리`보다 높았고, 별도 비교에서도 `운세·타로·심리상담·심리테스트`보다 높았다. 이는 절대 월간 검색량이 아니라 비교군 안의 상대지수지만, **불안을 다루는 익숙한 입구로 사주가 심리상담보다 강하다**는 방향을 보여준다.

동시에 해외에서는 Finch, Voidpet, Kinder World, Amaru처럼 감정과 셀프케어를 펫·생명체·식물에 맡기는 앱이 큰 사용 신호를 만들었다. 기회는 사주를 한 번 더 풀이하는 것이 아니라 **사주의 친숙한 입구와 비의존적 동반자의 위로 경험을 결합하는 것**이다.

## 2. 타겟과 사용 순간

### 초기 페르소나

**서연 / 27세 / 직장인**

- 사주·MBTI·타로를 재미와 자기이해 목적으로 가끔 본다.
- 답장이 늦거나 업무 피드백을 받은 밤에 생각이 많아진다.
- 병원이나 전문 상담이 필요한 상태는 아니지만 즉시 마음을 진정시키고 싶다.
- 친구에게 같은 걱정을 반복해서 말하는 것은 미안하고, 긴 일기는 귀찮다.
- 귀여운 캐릭터와 짧은 의식에는 거부감이 없다.

### 대상이 아닌 사람

- 자해·자살·타해·학대·급성 공황 등 즉각적인 전문 대응이 필요한 사람
- 정신건강 진단·치료 또는 약물 조언을 원하는 사람
- 만 19세 미만
- 사주 결과를 사실이나 중요한 의사결정 근거로 요구하는 사람

### Job Story

When **불안이 올라왔지만 상담을 받거나 지인에게 연락하기는 부담스러울 때**,
I want to **걱정을 판단하지 않는 작은 존재에게 잠시 맡기고 짧은 행동으로 끝내고 싶다**,
so I can **생각을 한 칸 내려놓고 원래 하던 일이나 휴식으로 돌아갈 수 있다**.

### Value Proposition

- **Who**: 사주·MBTI 같은 가벼운 자기이해 콘텐츠를 즐기는 일상 불안 사용자
- **Why**: 분석보다 즉시 이해받는 느낌과 짧은 종료가 필요하다.
- **What before**: AI와 끝없이 대화하거나 운세를 반복 확인하고, 명상 콘텐츠를 고르다 포기한다.
- **How**: 한 줄 걱정을 수호동물에게 맡기고 감정 이름 붙이기와 60~90초 행동을 수행한다.
- **What after**: 걱정은 사라지지 않아도 지금 할 행동과 종료점이 생긴다.
- **Alternatives**: 사주·운세 앱, ChatGPT, Wysa, Calm, Finch. 마음수호신은 예언·무한 채팅·콘텐츠 탐색 없이 한 번의 짧은 의식으로 끝난다.

## 3. 임팩트 가설

### 핵심 성공 지표

**활성화한 사용자 중 7일 안에 걱정 맡기기 의식을 3회 이상 끝낸 비율**

단순 앱 실행이나 체류시간은 성공으로 보지 않는다. 사용자가 의식을 끝내고 화면을 떠날 수 있어야 한다.

### 선행 지표

1. 수호동물 생성 후 첫 걱정 의식 완료율
2. 의식 전후 사용자가 직접 기록한 불안 강도 변화
3. 다음 날 재사용률과 7일 내 3회 완료율
4. 7일 마음흐름 해석 사전결제율
5. 수호동물 결과 공유 → 수신자 생성 → 첫 의식 완료 전환

### 정성적 임팩트

> “정답을 들은 건 아닌데, 누가 잠깐 같이 있어줘서 이제 그만 생각해도 될 것 같아요.”

### 성장 지표와 K

`K = 수호동물 공유 수 × 수신자 오픈율 × 수신자 첫 의식 완료율`

- 걱정 원문과 불안 강도는 공유하지 않는다.
- 수호동물 이름·성격·움직임만 선택적으로 공유한다.
- 실제 연쇄 전환을 두 코호트 이상 측정하기 전에는 high-K라고 표현하지 않는다.

## 4. 솔루션

### 형태 선택

설치 없이 링크로 시작하는 모바일 퍼스트 웹 앱으로 만든다. 4주 안에는 Finch 같은 육성 게임을 만들지 않고, 한 동반자의 세 상태와 짧은 의식만 구현한다.

### 핵심 가치 한 문장

> 사주로 만난 작은 수호동물에게 걱정 한 줄을 맡기고, 90초 안에 함께 내려놓는 가벼운 위로 의식.

### Must-have 1. 나만의 수호동물

- 생년월일과 가벼운 질문 1개로 목·화·토·금·수에서 영감을 받은 수호동물 하나를 배정한다.
- 결과는 오락·상징적 자기성찰이며 심리 진단이나 과학적 사실이 아님을 명시한다.
- 수호동물은 `기다림 / 함께함 / 쉬기` 세 상태만 가진다.
- 사용자가 오랫동안 오지 않아도 굶거나 아프거나 슬퍼지지 않는다.

### Must-have 2. 60~90초 걱정 맡기기

1. 사용자가 걱정 한 문장과 현재 강도 0~10을 입력한다.
2. AI는 감정 이름 하나와 판단 없는 문장 최대 두 개만 생성한다.
3. 사용자는 호흡, 손 펴기, 물 한 모금, 시선 돌리기, 한 문장 적기 중 하나를 60초간 수행한다.
4. 수호동물이 걱정을 옆에 내려놓고 함께 쉬는 장면으로 끝난다.
5. 사용자는 강도를 다시 고르고 앱을 닫는다.

무한 답장 입력창과 “더 물어보기” 버튼은 제공하지 않는다.

### Must-have 3. 조용한 정원과 유료 심화 가설

- 완료한 의식은 원문 없이 날짜·감정 이름·작은 행동만 정원에 흔적으로 남는다.
- 7일이 쌓이면 반복 감정, 자주 도움이 된 행동, 스스로 남긴 회복 문장을 묶은 `내 마음 사용설명서` 미리보기를 제공한다.
- 즉시 위로와 위기 안내는 항상 무료다.
- 전체 사용설명서 일회성 3,900원은 실제 가격이 아니라 사전결제 검증 가설이다.

### Non-goals

- 정확한 사주풀이, 미래 예언, 궁합·택일·오늘의 운세
- 여러 펫 수집, 먹이, 죽음, 가챠, 재화, 출석 스트릭
- 무한 AI 채팅, AI 연애·친구 관계, 양방향 음성 통화
- 공개 피드, 익명 인간 매칭, 팔로우·좋아요·DM
- 정신건강 진단·치료·상담 대체와 미성년자 지원
- 실제 PG 결제와 네이티브 앱

### 데모 흐름

1. 사용자가 생년월일과 질문 하나를 입력해 수호동물을 만난다.
2. 걱정 한 줄과 강도 7을 입력한다.
3. 수호동물이 걱정을 맡고 감정 이름과 짧은 위로를 건넨다.
4. 사용자가 60초 행동을 끝내고 강도를 5로 다시 기록한다.
5. 정원에 흔적이 남고 7일 마음 사용설명서 미리보기가 보인다.

## 5. B2C 수익과 성장

### 무료

- 수호동물 생성
- 하루 한 번 걱정 맡기기와 위기 안내
- 최근 7일의 원문 없는 정원
- 수호동물 소개 공유

### 유료 가설

| 상품 | 가설 가격 | 소비자가 사는 결과 |
|---|---:|---|
| 내 마음 사용설명서 | 3,900원 일회성 | 7일 패턴, 도움이 된 행동, 다음 7일 개인 의식 |
| 30일 정원 보관 | 사용설명서 포함 | 과거 흐름과 회복 문장 보관 |

구독, 젤리, 걱정 한 통당 과금은 MVP에서 테스트하지 않는다. 불안한 순간의 즉시 위로를 잠그지 않는다.

### 성장 루프

무료 수호동물 생성 → 결과 공유 → 친구도 수호동물 생성 → 첫 걱정 의식 → 7일 정원 → 마음 사용설명서 → 선택적 수호동물 공유

공유가 일어나지 않아도 제품 가치가 유지돼야 한다.

## 6. 리스크와 가정

### VUVF 가정

| 구분 | 가장 큰 가정 |
|---|---|
| Value | 작은 동반자와 의식이 같은 문장의 일반 AI 답변보다 더 위로되고 다시 쓰고 싶다. |
| Usability | 불안한 순간에도 한 줄 입력과 90초 행동을 끝낼 수 있다. |
| Viability | 무료 즉시 위로 뒤 7일 패턴 결과물에는 실제 돈을 낸다. |
| Feasibility | 안전 분류, 짧은 생성문, 세 상태 애니메이션을 4주 안에 안정적으로 구현한다. |

### 가장 위험한 가정

1. 펫이 귀엽다는 호감이 아니라 실제로 불안을 내려놓는 행동과 재사용을 만든다.
2. 사용자는 위로는 원하지만 유료 패턴 보고서에는 결제하지 않을 수 있다.

### 1주차 코드 없는 실험

#### 참가자

- 최근 일주일 일상 불안을 3회 이상 느낀 성인 12명
- 사주·MBTI·타로 중 하나를 최근 3개월 안에 사용한 사람
- 위기·진단·치료가 필요한 참여자는 모집하지 않는다.

#### 방법

- 같은 핵심 문장을 두 형식으로 순서를 바꿔 제공한다.
  - A: 일반 AI의 즉시 텍스트 위로
  - B: 작은 수호동물이 걱정을 맡고 60초 행동 뒤 쉬는 장면
- 5일 동안 참가자가 실제 불안한 순간에 최대 하루 한 번 사용한다.
- 운영자가 3분 안에 정해진 안전 템플릿으로 수기 응답한다.
- 사용 전후 강도, 이해받은 느낌, 행동 완료, 다음 날 재사용을 기록한다.
- 3일차부터 7일 마음 사용설명서 3,900원 환불 가능 사전결제를 제안한다.

#### 통과 기준

- 12명 중 8명 이상이 첫 의식을 끝낸다.
- 12명 중 7명 이상이 A보다 B를 선호한다.
- 12명 중 6명 이상이 5일 동안 3회 이상 다시 사용한다.
- 완료 세션의 60% 이상에서 강도가 1점 이상 낮아진다.
- 2명 이상이 마음 사용설명서를 사전결제한다.
- 최소 3명이 수호동물 결과를 실제 공유한다.

#### Kill / Pivot 기준

- B 선호가 6명 이하이면 펫·의식 가설을 폐기한다.
- 3회 재사용자가 5명 이하이면 독립 앱을 중단하고 일회성 콘텐츠로 축소한다.
- 사전결제 0명이면 유료 보고서 모델을 폐기한다.
- 2명 이상이 수호동물 때문에 죄책감·의존·불안 증가를 보고하면 펫 상태와 언어를 전면 재설계하고 구현을 중단한다.
- 위기 문장이 생성형 위로나 결제 화면으로 통과한 사례가 한 건이라도 있으면 공개 MVP를 중단한다.

### 안전 가드레일

- 자해·자살·타해·학대·급성 위기 신호에는 캐릭터 연출·생성·보상·결제를 중단한다.
- 한국에서는 자살예방상담전화 109, 즉각 위험 시 119·112, 신뢰하는 사람과 전문가 연결을 직접 안내한다.
- “나만 네 마음을 안다”, “떠나지 마”, “매일 와야 한다” 같은 의존 문구를 금지한다.
- 수호동물은 절대 굶거나 죽거나 사용자의 부재를 탓하지 않는다.
- 걱정 원문은 기본 최소 저장하고 사용자가 즉시 삭제할 수 있게 한다.
- 사주는 오락·상징임을 고지하고 건강·돈·관계의 결정을 지시하지 않는다.
