## T1. 한입코치 — 첫 번째 가치·결제 검증

### 준비

- [ ] 최근 한 달 동안 식단 기록을 두 번 이상 포기한 성인 10명 모집
- [ ] 의료적 식단·섭식장애 위험·미성년 제외 문구 확인
- [ ] 중립형과 단호형 피드백 템플릿 작성
- [ ] 사진 삭제 시점과 동의 문구 작성
- [ ] 기간형 유료 코칭 가짜 문 또는 실제 예약금 링크 준비

### 실행

- [ ] 5일 동안 하루 한 끼 사진 수신
- [ ] 5분 안에 `관찰 → 단호한 반응 → 다음 행동 하나` 수기 회신
- [ ] 다음 끼니에서 행동을 지켰는지 확인
- [ ] 중립형과 단호형의 전송 지속·행동 실행 차이 기록
- [ ] 3일차 이후 실제 결제 행동 측정

### 판정

- [ ] 3일 이상 사진 전송률 확인
- [ ] 다음 행동 2회 이상 실행률 확인
- [ ] 수치심·불안·회피 보고 확인
- [ ] 실제 결제 수 확인
- [ ] Go / Tone Pivot / Kill 결정 기록

## T2. 마음수호신 — 동반자·의식·유료 심화 검증

### 준비

- [ ] 최근 일주일 동안 일상 불안을 세 번 이상 느낀 성인 12명 모집
- [ ] 최근 3개월 안에 사주·MBTI·타로 중 하나를 사용한 사람으로 제한
- [ ] 임상적 불안·위기 상황·미성년 제외 및 도움 안내 문구 작성
- [ ] 걱정 한 문장을 받는 동의·삭제 규칙 작성
- [ ] 수호동물 5종과 `기다림 / 함께함 / 쉬기` 세 상태 준비
- [ ] `감정 이름 → 짧은 위로 → 60초 행동 → 함께 쉬기` 수기 템플릿 작성
- [ ] 7일 마음 사용설명서의 환불 가능한 사전결제 링크 준비

### 실행

- [ ] 5일 동안 불안한 순간 걱정 한 문장과 강도 수신
- [ ] 같은 문장을 일반 AI 텍스트 A와 수호동물 의식 B로 순서를 바꿔 제공
- [ ] 3분 안에 수호동물과 60초 의식 수기 제공
- [ ] 사용 전후 불안 강도와 작은 행동 완료 측정
- [ ] 다음 날 다시 사용했는지 측정
- [ ] 7일 패턴 보고서 사전결제 행동 측정

### 판정

- [ ] A보다 수호동물 의식 B를 선호한 사용자 수 확인
- [ ] 불안 강도 변화와 3분 행동 완료율 확인
- [ ] 3회 이상 재사용한 사용자 수 확인
- [ ] 사전결제와 펫 죄책감·불안 악화·의존 징후 확인
- [ ] 수호동물 앱 / 일회성 사주 콘텐츠 / Kill 결정 기록
