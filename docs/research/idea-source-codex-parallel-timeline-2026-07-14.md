## 5분 체크포인트

매 체크마다 다음을 확인한다.

1. 네 runner PID가 살아 있는가
2. 각 runner가 자기 lane 밖의 배치를 쓰지 않는가
3. `passed` 배치 수와 최근 통과 배치가 늘었는가
4. partial JSONL 행 수와 수정 시간이 움직이는가
5. Codex CLI 오류·타임아웃·구조화 출력 실패가 있는가
6. 15분 동안 한 lane의 파일 변화가 없으면 해당 lane만 중단된 지점에서 재개한다

재개할 때 기존 JSONL을 지우거나 `--replace-local`을 사용하지 않는다. 이미 검증된 행은 그대로 두고 누락 키만 다시 요청한다.
