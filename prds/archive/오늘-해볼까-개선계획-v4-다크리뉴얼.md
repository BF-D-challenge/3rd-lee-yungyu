### 3.3 인트로 — toss.im/simplicity 첫 진입 시퀀스 이식 (프레임 실측 기반)

> 실측 근거: [`docs/references/simplicity-intro/`](../../docs/references/simplicity-intro/README.md) (2026-07-06 CDP 40프레임 + 뮤테이션 5,514건). 토스 인트로의 핵심은 ① 검정 위 **`mix-blend-mode: lighten` 빛줄기 스트립**이 워드를 관통하는 글리치, ② **인트로의 끝이 곧 첫 화면**(워드마크가 9:16 카드로 축소되며 캐러셀에 안착하는 원 컨티뉴어스 샷).

우리 버전 4단계 (총 ~4.5s, 골드/퍼플로 색만 치환):

| 단계 | 시간 | 토스 원본 | 오늘 해볼까 |
|---|---|---|---|
| A. 워드 등장 | 0~1.2s | 검정 화면에 TOSS·DESIGN·CONFERENCE 자간 넓게 순차 등장 | `#0a0a0b` 화면에 `오늘` `해볼까` (Playfair, 자간 넓게) 순차 페이드인 |
| B. 빛줄기 스윕 | 0.6~2.4s | 파란 빛줄기 비디오 스트립(1440×150) lighten 블렌드 | **비디오 없이** CSS 그라디언트 스트립(투명→골드 #c9a962→퍼플 #b288f6→투명) + `mix-blend-mode: lighten` + translateX 스윕 2회 |
| C. 워드마크 블룸 | 2.4~3.3s | Simplicity 수렴 + 파란 라디얼 블룸 | 두 워드가 붙으며 `오늘 해볼까` 워드마크 + 화면 가장자리 골드 라디얼 블룸 |
| D. 카드 안착 | 3.3~4.5s | 풀블리드 → 센터 9:16 카드로 축소, 캐러셀 드러남 | 풀블리드 → **9:16 카드로 축소되면서 그 뒤에서 부채꼴 덱이 펼쳐지고** 랜딩 히어로(카피+CTA+카운터)가 드러남 |

규칙: 아무 클릭/스크롤/키 입력 시 즉시 스킵(랜딩 최종 상태로 점프). `prefers-reduced-motion`이면 인트로 생략하고 정적 히어로. `sessionStorage`로 세션당 1회만. 애니메이션은 `opacity/transform`만(레이아웃 속성 금지, 토스와 동일 원칙).

## 4. 화면별 ASCII 목업 (신입용 — 이대로 만들면 됨)

> 모바일 375px 기준 세로 1열. `▓`=블러, `∙`=muted 텍스트, `[ ]`=버튼/칩, `†`=골드 액센트.

```tsx
  { key: 'words',  at: 0 },     // A: '오늘' '해볼까' 순차 등장
  { key: 'streak', at: 600 },   // B: 골드 빛줄기 스윕
  { key: 'bloom',  at: 2400 },  // C: 워드마크 수렴 + 라디얼 블룸
  { key: 'settle', at: 3300 },  // D: 카드로 축소 + 부채꼴/히어로 드러남
  { key: 'done',   at: 4500 },
] as const

export function Intro({ onDone }: { onDone: () => void }) {
  const [phase, setPhase] = useState<typeof PHASES[number]['key']>('words')

  useEffect(() => {
    // 세션 1회 + reduced-motion이면 즉시 스킵
    if (sessionStorage.getItem('introSeen') ||
        matchMedia('(prefers-reduced-motion: reduce)').matches) return skip()
    const timers = PHASES.map(p => setTimeout(() =>
      p.key === 'done' ? skip() : setPhase(p.key), p.at))
    // 어떤 입력이든 즉시 스킵 (토스와 동일)
    const skipEvents = ['pointerdown', 'wheel', 'keydown'] as const
    skipEvents.forEach(e => addEventListener(e, skip, { once: true }))
    return () => { timers.forEach(clearTimeout); skipEvents.forEach(e => removeEventListener(e, skip)) }
    function skip() { sessionStorage.setItem('introSeen', '1'); onDone() }
  }, [onDone])

  return (
    <div data-intro className="fixed inset-0 z-50 bg-[--bg] grid place-items-center">
      <h1 className={`serif text-[--ink] tracking-[.4em] transition-all duration-700
        ${phase === 'words' ? 'opacity-100' : ''}
        ${phase === 'bloom' || phase === 'settle' ? 'tracking-normal scale-125' : ''}`}>
        오늘 해볼까
      </h1>
      {phase === 'streak' && <div className="intro-streak" />}
      {phase === 'bloom' && <div className="absolute inset-0"
        style={{ boxShadow: 'inset 0 0 180px 40px rgba(201,169,98,.25)' }} />}
      {phase === 'settle' && <div className="absolute aspect-[9/16] w-[300px] rounded-[--r-card]"
        style={{ animation: 'settle-to-card .9s cubic-bezier(.65,0,.35,1) both' }} />}
    </div>
  )
}
```

### 6.3 씨앗-conditioned 뽑기 엔진 — `lib/draw.ts` (R12·R14 핵심)

```ts
import combos from '@/app/data/combos.json'
// combos.json 스키마:
// { seeds: { [seedTag]: { pains: string[], forms: string[] } },   ← allowlist (R12)
//   golden: { seedTag, pain, form, sentence }[] }                 ← 사전검수 골든 (R14)

const recent: string[] = []   // 세션 내 중복 회피 큐 (R13)

export function draw(seedTag: string, spinIndex: number, locked: { pain?: string; form?: string }) {
  const pool = combos.seeds[seedTag]
  if (!pool) throw new Error(`unknown seed: ${seedTag}`) // 자유입력은 최근접 태그 매핑 후 진입 (R16)

  // 첫 2~3스핀은 골든 조합 우선 — 신뢰 확보 후 랜덤 개방 (R14)
  const goldens = combos.golden.filter(g => g.seedTag === seedTag && !recent.includes(key(g)))
  if (spinIndex < 3 && goldens.length) return remember(goldens[spinIndex % goldens.length])

  // 리롤: 고정 안 한 릴만, allowlist 안에서만 (락 = 앵커 유지)
  const pick = <T,>(xs: T[]) => xs[Math.floor(Math.random() * xs.length)]
  let combo
  do {
    combo = {
      seedTag,
      pain: locked.pain ?? pick(pool.pains),
      form: locked.form ?? pick(pool.forms),
    }
  } while (recent.includes(key(combo)) && recent.length < pool.pains.length * pool.forms.length)
  return remember(combo)
}

const key = (c: { seedTag: string; pain: string; form: string }) => `${c.seedTag}|${c.pain}|${c.form}`
function remember<T extends { seedTag: string; pain: string; form: string }>(c: T): T {
  recent.push(key(c)); if (recent.length > 30) recent.shift()
  return c
}
```

### 6.4 결과 문장 템플릿 + 조사 자동교정 — `lib/josa.ts` (정적 데모 v4에서 이식)

```ts
// 받침 유무로 조사 선택: '러닝' + 을/를 → '러닝을', '요가' → '요가를'
export function josa(word: string, pair: '을/를' | '이/가' | '은/는' | '으로/로') {
  const last = word.charCodeAt(word.length - 1)
  const hasBatchim = (last - 0xac00) % 28 > 0
  const [a, b] = pair.split('/')
  if (pair === '으로/로') return word + ((last - 0xac00) % 28 === 8 ? b : hasBatchim ? a : b) // ㄹ받침 예외
  return word + (hasBatchim ? a : b)
}

export const sentence = (c: { seed: string; pain: string; form: string }) =>
  `${josa(c.seed, '을/를')} 좋아하는 사람 중 ${c.pain} 사람을 위한 ${c.form}`
```

### 6.5 슬롯 상태 (락+리롤+캡) — `lib/slot-store.ts` (zustand)

```ts
import { create } from 'zustand'
import { draw } from './draw'

type SlotState = {
  seedTag: string; spins: number; combo: ReturnType<typeof draw> | null
  locked: { pain: boolean; form: boolean }
  spin: () => void; toggleLock: (reel: 'pain' | 'form') => void
}
export const useSlot = create<SlotState>((set, get) => ({
  seedTag: '', spins: 0, combo: null, locked: { pain: false, form: false },
  spin: () => {
    const { seedTag, spins, combo, locked } = get()
    if (spins >= 5) return track('spin_cap_hit', { daily_count: spins }) // → S1b 페이월
    const next = draw(seedTag, spins, {
      pain: locked.pain ? combo?.pain : undefined,   // 🔒 고정 릴은 유지 (v7 규칙)
      form: locked.form ? combo?.form : undefined,
    })
    set({ combo: next, spins: spins + 1 })
    track('slot_spin', { spin_index: spins + 1, seed_tag: seedTag, ...next })
  },
  toggleLock: (reel) => {
    set(s => ({ locked: { ...s.locked, [reel]: !s.locked[reel] } }))
    track('reel_lock_toggle', { reel })
  },
}))
```
