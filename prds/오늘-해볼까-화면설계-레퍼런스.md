# 오늘 해볼까 — 화면 설계 레퍼런스 (mobbin, 2026-07-07 · 예쁜 앱 큐레이션)

> 칭찬받기 피벗([피벗 계획](오늘-해볼까-칭찬받기-피벗.md))의 각 화면을 **시각적으로 뛰어난 실제 앱 패턴에서 출발**시키기 위한 mobbin 레퍼런스. 우리 스킨(다크 Aurora 글래스·미니멀 센터 컬럼)으로 번역하되, 레이아웃·인터랙션 문법은 아래를 기준으로 한다.
> ※ 정보 밀도만 높고 시각 완성도 낮은 레퍼런스(예: Taobao 扔有家)는 제외.

---

## S5 — 수신자 응원 카드 (단일, 무로그인) · /vs A/B 응원 대결

**★ 정석 레퍼런스 = Gas 원본** (우리 피벗의 직계 조상):
- [Gas 투표 — 메달 + 4명 2×2](https://mobbin.com/screens/76dcace8-eb77-4f55-9ee7-43200e682f8d) — 솔리드 컬러 풀블리드 배경 + 상단 이모지/메달 + 긍정 프롬프트 + **친구 4명 2×2 버튼**(탭 시 좌→우 컬러 필 애니메이션) + "Tap to continue".
- [Gas 투표 — 이모지 프롬프트](https://mobbin.com/screens/9742042a-6968-4b19-9409-84decde1274a) — "Cute nerd tbh" 큰 이모지 + 4택 + **Shuffle · Skip**. 프롬프트 자체가 주인공, UI는 극단적으로 미니멀.

**보조 — 예쁜 폴/스토리 스티커**:
- [Instagram Story 폴 스티커 "IF I WERE A CELEBRITY"](https://mobbin.com/screens/130d427a-9c21-46ff-9dc2-62ae1e7fece6) — 그라데이션 헤더 + A/B 알약 2개. 우리 /vs 대결의 **가벼운 톤**.
- [Peanut 폴 결과(퍼센트 바)](https://mobbin.com/screens/3ea35c9f-aa47-42c7-9161-68533db66b43) — 라운드 알약 + % 바 + 코멘트 스레드. 응원 후 집계 표시.

→ **적용**: 우리 응원 칩(🔥/🙌/👀/💪)을 Gas식 **2×2 큰 버튼 + 탭 컬러 필**로. 다크 Aurora 배경 위 풀블리드. 프롬프트("어느 쪽을 응원할래?")를 주인공으로, 카운트는 선택 후에만. Shuffle/Skip 대신 "나도 뽑아보기".

## S3 — 발행 9:16 공유 카드 (브랙 카드)

**★ 다크 톤 그라데이션 카드**:
- [QUITTR Progress Card](https://mobbin.com/screens/276fa4dc-c173-4e23-9305-8fa75f2e96a7) ★★ — **별밤 배경 위 오렌지→시안 대각 그라데이션 세로 카드** + 상단 로고/북마크 + "Active Streak 6 days" + 하단 다크 바(Name/Free since). **우리 다크 Aurora 카드에 그대로 이식 가능한 톤.**
- [Beli "Top 10% Diner"](https://mobbin.com/screens/b486127e-a01f-4659-9abb-74b16f1f69b5) ★ — 딥네이비 카드 + **큰 숫자 2개(20 Restaurants · 18 Cuisines)** + 세리프 워드마크 + "Share this page" 채널 행(IG Story/Post/TikTok/Messages). 우리 브리프 요약 지표 배치.
- [Calm Day Streak](https://mobbin.com/screens/75c25c9b-df74-43f1-baea-e61def7bb524) — 딥블루 카드 + 리본 배지 + 3지표 + 세리프 워드마크 + "Send Message / Share" 2버튼.

**바이럴 초대 순간**:
- [IRL "You're on a roll ♥ Share your accomplishments"](https://mobbin.com/screens/46c8a2e3-abb5-4d59-afb8-51b850f205c8) ★ — 코랄→퍼플 그라데이션 카드 + 채널 아이콘 5개 + "Share / Remind me later". Gas식 발행 직후 감정 후크의 **예쁜 버전**(Peanut보다 세련).
- [Duolingo Year-in-Review 공유](https://mobbin.com/screens/81b67776-4a5d-40d9-860e-9b3b4122357a) — 카드 스와이프 + "SHARE FOR A REWARD". 공유에 인센티브.

→ **적용**: 9:16 카드 = QUITTR 그라데이션 톤 + Beli 지표 배치 + 세리프 워드마크("오늘 해볼까"). 발행 성공 시 IRL식 "오늘 만들 거 정했다!" 초대 카드.

## S6 — 대시보드 + 리빌 (결제)

**★ "누가 나를 —" 리빌** (구독 아닌 일회성으로 번역):
- [Hinge "Send and see all the likes you want"](https://mobbin.com/screens/ab3d49ac-4e97-4037-ab8d-8ef7bded5ece) ★ — 히어로 이미지 + 혜택 리스트(무한대/전원 공개 아이콘) + 가격 카드. **Tinder보다 세련된 리빌 페이월.** 우리는 가격을 일회성 패스로.
- [Bumble "See who likes you" (Beeline)](https://mobbin.com/screens/933bf631-efe8-42b8-90ed-3d172e39eaff) — 골드 톤 + 겹친 블러 아바타 3개 + "Unlock Beeline". 블러 리빌의 정석.
- [Coffee Meets Bagel "Activity Report" 잠금 타일](https://mobbin.com/screens/cb679aa2-1632-4830-bd55-dc3ec03b4433) ★ — **블루 카드 안 2×2 Locked 지표 + "Upgrade now"**. 우리 무료 집계→유료 상세의 카드 문법.

**무료 층 카운트 타일**:
- [Beli "Top 10% Diner" 숫자 타일](https://mobbin.com/screens/b486127e-a01f-4659-9abb-74b16f1f69b5) — 큰 숫자 2개 문법을 "응원 N명 / 🔥 N명"에.

→ **적용**: 무료 = Beli/Universe식 큰 카운트 타일. 유료 리빌 = Hinge식 혜택 + 일회성 가격(구독·스크램블 금지, 피벗 §4). "응원만 N명·🔥 0명 = 강한 보류" 케이스 명시.
